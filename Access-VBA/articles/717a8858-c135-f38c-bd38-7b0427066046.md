
# Synchronizer has not been set up. (Error 3543)

 **Last modified:** December 30, 2015

 _ **Applies to:** Access 2013 | Access 2016_

The Synchronizer for your computer has not been set up.

 To configure the Synchronizer


1. Start Replication Manager.
    
2. On the  **Tools** menu, click **Configure Microsoft Replication Manager**.
    
3. Follow the instructions in the wizard.
    

If you are using a third-party application, obtain the Synchronizer Setup disk from the developer supporting your application.
