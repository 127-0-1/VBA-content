
# GroupCriterion2.GroupOn Property (Project)

Gets or sets the type of grouping for a field used as a criterion in a group definition. Read/write  **Long**. Can be one of the **[PjGroupOn](72b06ac4-6028-2845-7bac-d4cde3c79c33.md)** constants.


## Syntax

 _expression_. **GroupOn**

 _expression_ An expression that returns a **GroupCriterion2** object.
