
# ChartArea.Height Property (Word)

Returns or sets the height, in points, of the object. Read/write  **Double** .


## Syntax

 _expression_ . **Height**

 _expression_ A variable that represents a **[ChartArea](7b3384df-f331-033d-4dfa-ee2ff26111c6.md)** object.
