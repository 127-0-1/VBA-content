
# MailMergeFields.Item Method (Word)

Returns an individual  **MailMergeField** object in a collection.


## Syntax

 _expression_ . **Item**( **_Index_** )

 _expression_ Required. A variable that represents a **[MailMergeFields](9d2dfd45-c52b-500e-15bf-1e678e6c1e92.md)** collection.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _Index_|Required| **Long**|The individual object to be returned. Can be a  **Long** indicating the ordinal position of the individual object.|

### Return Value

MailMergeField
