
# AutoCaption.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **AutoCaption** object.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents an **[AutoCaption](895b5181-d36f-7f63-572a-c2d37c878e17.md)** object.
