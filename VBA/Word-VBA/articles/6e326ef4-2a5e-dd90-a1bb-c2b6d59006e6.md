
# Rows.Count Property (Word)

Returns a  **Long** that represents the number of rows in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents a **[Rows](cd83d0ef-f743-1886-54de-497017c5f542.md)** collection.
