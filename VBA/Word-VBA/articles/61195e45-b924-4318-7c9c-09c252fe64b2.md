
# Task.Name Property (Word)

Returns the name of the specified object. Read-only  **String** .


## Syntax

 _expression_ . **Name**

 _expression_ Required. A variable that represents a **[Task](8802fcd5-0947-2ea0-308a-376077633e34.md)** object.
