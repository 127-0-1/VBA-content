
# MailMerge.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **MailMerge** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[MailMerge](b228c4d6-9ca7-8795-12f6-d32e62844a83.md)** object.
