
# ChartGroup.Parent Property (Word)

Returns the parent for the specified object. Read-only  **Object** .


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[ChartGroup](ea5a2610-9c00-9c95-8366-f9b0fcdf90be.md)** object.
