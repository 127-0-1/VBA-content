
# Application.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Application** object.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents an **[Application](d1cf6f8f-4e88-bf01-93b4-90a83f79cb44.md)** object.
