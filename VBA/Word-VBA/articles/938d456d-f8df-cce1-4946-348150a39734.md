
# DownBars.Parent Property (Word)

Returns the parent for the specified object. Read-only  **Object** .


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[DownBars](d0cf170e-0c58-2d01-a4b2-1eaf65dbfa3c.md)** object.
