
# FootnoteOptions.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **FootnoteOptions** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[FootnoteOptions](5fdeb6d6-ce33-44f5-62c1-743fc3770457.md)** collection.
