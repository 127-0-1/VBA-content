
# XMLNode.Delete Method (Word)

Deletes the specified XML element from an XML document.


## Syntax

 _expression_ . **Delete**

 _expression_ Required. A variable that represents a **[XMLNode](fe305ba9-7375-ad4f-6036-155add17a9d0.md)** object.
