
# OMath.ParentRow Property (Word)

Returns an  **[OMathMatRow](a6e96db5-9119-51a2-f303-a6c74b5be88d.md)** object that represents the parent row in a matrix. Read-only.


## Syntax

 _expression_ . **ParentRow**

 _expression_ An expression that returns an **[OMath](82f2f81b-e2d5-140f-bdcc-8b52b821b24d.md)** object.
