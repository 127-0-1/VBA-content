
# Document.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Document** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[Document](8d83487a-2345-a036-a916-971c9db5b7fb.md)** object.
