
# Style.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Style** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[Style](473f8f41-2cba-769e-c0da-441d9d85b009.md)** object.
