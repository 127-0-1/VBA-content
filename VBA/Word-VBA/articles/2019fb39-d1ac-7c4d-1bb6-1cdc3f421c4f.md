
# TextEffectFormat.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **TextEffectFormat** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[TextEffectFormat](b274e5be-ed5b-7d63-aa4b-1d67b63e7c0b.md)** object.
