
# Trendline.Border Property (Word)

Returns the border of the object. Read-only  **[ChartBorder](eea90670-c599-2ec8-5b7b-c946a4bcd638.md)** .


## Syntax

 _expression_ . **Border**

 _expression_ A variable that represents a **[Trendline](1cfe897f-26ad-a838-ed9b-f3fd945ff7ea.md)** object.
