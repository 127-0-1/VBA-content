
# Template.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Template** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[Template](47d1d92d-bba9-3f2a-9c71-22ac43159bd3.md)** object.
