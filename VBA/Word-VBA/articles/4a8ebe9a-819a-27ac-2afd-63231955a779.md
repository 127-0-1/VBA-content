
# LegendKey.Shadow Property (Word)

Returns or sets a value that indicates whether the object has a shadow. Read/write  **Boolean** .


## Syntax

 _expression_ . **Shadow**

 _expression_ A variable that represents a **[LegendKey](07578528-3e73-7898-47dc-296aefb854f0.md)** object.
