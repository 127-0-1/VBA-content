
# Frame.Width Property (Word)

Returns or sets the width (in points) of the frame, in points. Read/write  **Long** .


## Syntax

 _expression_ . **Width**

 _expression_ A variable that represents a **[Frame](d36d3361-9e93-7dd9-b8c9-0ce503e03810.md)** object.
