
# Field.Index Property (Word)

Returns a  **Long** that represents the position of an item in a collection. Read-only.


## Syntax

 _expression_ . **Index**

 _expression_ Required. A variable that represents a **[Field](75139aa4-89f4-2ffb-b964-8dc805b9a32b.md)** object.


## Example

This example returns the position of the selected field in the Fields collection.


```
num = Selection.Fields(1).Index
```
