
# Pane.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Pane** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[Pane](4a0c2690-d9d2-4e34-fef4-cc41365f5251.md)** object.
