
# Find.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Find** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[Find](da822788-cad5-992a-a835-18cc574cc324.md)** object.
