
# Application.Visible Property (Word)

 **True** if the specified object is visible. Read/write **Boolean** .


## Syntax

 _expression_ . **Visible**

 _expression_ Required. A variable that represents an **[Application](d1cf6f8f-4e88-bf01-93b4-90a83f79cb44.md)** object.


## Example

This example hides Microsoft Word.


```vb
Application.Visible = False
```
