
# Category.Index Property (Word)

Returns a  **Long** that represents the position of an item in a collection. Read-only.


## Syntax

 _expression_ . **Index**

 _expression_ Required. A variable that represents a **[Category](5485ae39-fbcf-b18f-b1f9-945e220ecd2a.md)** object.
