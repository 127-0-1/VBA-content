
# ChartTitle.Height Property (Word)

Returns the height, in points, of the object. Read-only.


## Syntax

 _expression_ . **Height**

 _expression_ A variable that represents a **[ChartTitle](fc8ca540-0a29-123b-2fdf-b16aaa1f940c.md)** object.


### Return Value

Double
