
# Categories.Count Property (Word)

Returns the number of items in the  **Categories** collection. Read-only **Long** .


## Syntax

 _expression_ . **Count**

 _expression_ A variable that represents a **[Categories](f5f5081d-4309-6617-28da-c369c1fe690c.md)** collection.
