
# ParagraphFormat.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **ParagraphFormat** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[ParagraphFormat](712d754a-dc92-f1a3-531d-dfae74a42c23.md)** object.
