
# ChartBorder.Parent Property (Word)

Returns the parent for the specified object. Read-only  **Object** .


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[ChartBorder](eea90670-c599-2ec8-5b7b-c946a4bcd638.md)** object.
