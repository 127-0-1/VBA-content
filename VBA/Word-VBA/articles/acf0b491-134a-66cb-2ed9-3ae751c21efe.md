
# ChartArea.Parent Property (Word)

Returns the parent for the specified object. Read-only  **Object** .


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[ChartArea](7b3384df-f331-033d-4dfa-ee2ff26111c6.md)** object.
