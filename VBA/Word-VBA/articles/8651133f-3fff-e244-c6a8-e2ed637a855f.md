
# Trendline.Name Property (Word)

Returns or sets name of the object. Read/write  **String** .


## Syntax

 _expression_ . **Name**

 _expression_ A variable that represents a **[Trendline](1cfe897f-26ad-a838-ed9b-f3fd945ff7ea.md)** object.
