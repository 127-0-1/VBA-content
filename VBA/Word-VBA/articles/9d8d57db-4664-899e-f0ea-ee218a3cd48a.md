
# Style.Locked Property (Word)

 **True** if a style cannot be changed or edited. Read/write **Boolean** .


## Syntax

 _expression_ . **Locked**

 _expression_ An expression that represents a **[Style](473f8f41-2cba-769e-c0da-441d9d85b009.md)** object.
