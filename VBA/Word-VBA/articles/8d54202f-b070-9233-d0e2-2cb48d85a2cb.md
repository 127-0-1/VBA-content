
# Documents.Count Property (Word)

Returns a  **Long** that represents the number of documents in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents a **[Documents](fc4ac973-19c1-703a-5538-f4426b8b7564.md)** collection.
