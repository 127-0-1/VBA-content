
# Borders.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Borders** collection.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[Borders](6dd1d4cc-2dcf-22c7-a299-4721a5543ba3.md)** collection.
