
# OMath.ConvertToMathText Method (Word)

Converts an equation to math text.


## Syntax

 _expression_ . **ConvertToMathText**

 _expression_ An expression that returns an **[OMath](82f2f81b-e2d5-140f-bdcc-8b52b821b24d.md)** object.
