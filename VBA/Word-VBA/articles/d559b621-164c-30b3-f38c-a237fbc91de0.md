
# DisplayUnitLabel.Width Property (Word)

Returns the width, in points, of the object. Read-only.


## Syntax

 _expression_ . **Width**

 _expression_ A variable that represents a **[DisplayUnitLabel](9b028f6c-fd66-f767-f3d1-09de0fbdc148.md)** object.
