
# ChartGroups.Parent Property (Word)

Returns the parent for the specified object. Read-only  **Object** .


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[ChartGroups](37136fbd-8740-c817-9666-993bc5d4c847.md)** object.
