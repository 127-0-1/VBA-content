
# Series.Shadow Property (Word)

Returns or sets a value that indicates whether the object has a shadow. Read/write  **Boolean** .


## Syntax

 _expression_ . **Shadow**

 _expression_ A variable that represents a **[Series](212c323f-8acb-2ba7-1359-ab0f43268e77.md)** object.
