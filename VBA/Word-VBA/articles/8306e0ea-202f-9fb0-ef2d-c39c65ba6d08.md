
# Trendlines.Count Property (Word)

Returns the number of objects in the collection. Read-only  **Long** .


## Syntax

 _expression_ . **Count**

 _expression_ A variable that represents a **[Trendlines](06c20a75-4afc-03f5-1eec-eee1559d3f52.md)** object.
