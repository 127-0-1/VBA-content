
# OMathDelim.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **OMathDelim** object.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[OMathDelim](6c203045-a0e0-6bf7-d8c7-01e991a3a168.md)** object.
