
# MappedDataField.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **MappedDataField** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[MappedDataField](35b9b770-bf18-8922-7c3a-431f454561e9.md)** object.
