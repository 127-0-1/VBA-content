
# CaptionLabel.Name Property (Word)

Returns the name of the specified object. Read-only  **String** .


## Syntax

 _expression_ . **Name**

 _expression_ Required. A variable that represents a **[CaptionLabel](71c82dfd-6a66-e0f4-e30f-ae453c764864.md)** object.
