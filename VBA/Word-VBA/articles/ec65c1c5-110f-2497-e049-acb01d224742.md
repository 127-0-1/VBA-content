
# UpBars.Border Property (Word)

Returns the border of the object. Read-only  **[ChartBorder](eea90670-c599-2ec8-5b7b-c946a4bcd638.md)** .


## Syntax

 _expression_ . **Border**

 _expression_ A variable that represents an **[UpBars](22dff1d2-8f1b-8c48-354c-570906e0f830.md)** object.
