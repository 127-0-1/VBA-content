
# AutoCaption.Index Property (Word)

Returns a  **Long** that represents the position of an item in a collection. Read-only.


## Syntax

 _expression_ . **Index**

 _expression_ Required. A variable that represents an **[AutoCaption](895b5181-d36f-7f63-572a-c2d37c878e17.md)** object.
