
# Rectangles.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Rectangles** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[Rectangles](c1de5e7f-13b1-e35a-d9f1-9a8f1246e2e5.md)** collection.
