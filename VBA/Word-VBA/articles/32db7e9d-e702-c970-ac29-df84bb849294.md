
# Footnote.Delete Method (Word)

Deletes the specified footnote.


## Syntax

 _expression_ . **Delete**

 _expression_ Required. A variable that represents a **[Footnote](877340c4-14f9-4560-eaf8-2c6482a1ade8.md)** object.
