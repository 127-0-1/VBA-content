
# GroupShapes.Count Property (Word)

Returns a  **Long** that represents the number of shapes in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents a **[GroupShapes](de29d571-476b-fa8b-619e-f7d0181d9756.md)** collection.
