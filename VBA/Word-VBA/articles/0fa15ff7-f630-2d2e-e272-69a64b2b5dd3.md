
# LegendKey.Height Property (Word)

Returns the height, in points, of the object. Read-only  **Double** .


## Syntax

 _expression_ . **Height**

 _expression_ A variable that represents a **[LegendKey](07578528-3e73-7898-47dc-296aefb854f0.md)** object.
