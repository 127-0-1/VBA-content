
# OMathEqArray.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **OMathEqArray** object.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[OMathEqArray](9d3516fd-7f08-eacc-71e3-c27e6e373857.md)** object.
