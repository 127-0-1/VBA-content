
# EmailSignatureEntry.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **EmailSignatureEntry** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents an **[EmailSignatureEntry](a8cf11de-7a46-c609-3cd7-508e9ef91e09.md)** object.
