
# Lists.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Lists** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[Lists](1fd927c5-6186-5ca0-80ae-c2ab225d092c.md)** collection.
