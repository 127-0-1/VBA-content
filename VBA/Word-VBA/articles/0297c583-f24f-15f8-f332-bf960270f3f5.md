
# View.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **View** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[View](8bf5b26b-14c0-1985-65b2-3e034360baeb.md)** object.
