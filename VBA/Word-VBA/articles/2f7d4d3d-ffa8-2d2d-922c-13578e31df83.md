
# Variable.Delete Method (Word)

Deletes the specified variable.


## Syntax

 _expression_ . **Delete**

 _expression_ Required. A variable that represents a **[Variable](e6a75f54-6f91-75b4-7ca0-9be302e8dbe0.md)** object.
