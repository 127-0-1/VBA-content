
# CustomLabel.Name Property (Word)

Returns or sets the name of the specified object. Read/write  **CustomLabel** .


## Syntax

 _expression_ . **Name**

 _expression_ Required. A variable that represents a **[CustomLabel](a89ff4e1-ff8a-8a8f-afa2-6071bb49355b.md)** object.
