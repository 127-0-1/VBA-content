
# AddIn.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **AddIn** object.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[AddIn](5615a8a9-1fd6-04fa-1fee-ec16502bd84a.md)** object.
