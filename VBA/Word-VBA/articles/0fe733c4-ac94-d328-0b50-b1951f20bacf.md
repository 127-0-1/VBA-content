
# Point.Width Property (Word)

Returns the width, in points, of the object. Read-only.


## Syntax

 _expression_ . **Width**

 _expression_ A variable that represents a **[Point](349ea9a3-9e9a-b26f-146f-799d39c3d4a9.md)** object.
