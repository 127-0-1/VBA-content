
# Indexes.Count Property (Word)

Returns a  **Long** that represents the number of indexes in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents an **[Indexes](0441446a-c1b5-d333-5950-906fe463b61d.md)** collection.
