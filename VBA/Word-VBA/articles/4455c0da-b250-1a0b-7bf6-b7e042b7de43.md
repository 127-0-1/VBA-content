
# Language.Name Property (Word)

Returns the name of the specified object. Read-only  **String** .


## Syntax

 _expression_ . **Name**

 _expression_ Required. A variable that represents a **[Language](0acc4a42-b4c2-a415-0e38-a049b085dc86.md)** object.
