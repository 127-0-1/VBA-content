
# MailMergeField.Copy Method (Word)

Copies the specified mail merge field to the Clipboard.


## Syntax

 _expression_ . **Copy**

 _expression_ Required. A variable that represents a **[MailMergeField](8beb6228-079c-008c-10aa-3f8f711fcf5c.md)** object.
