
# Editor.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Editor** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents an **[Editor](af0c80f5-8c8a-be0e-4475-d3b3b3bacd0d.md)** object.
