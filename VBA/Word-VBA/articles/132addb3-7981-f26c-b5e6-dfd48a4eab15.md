
# CustomProperty.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **CustomProperty** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[CustomProperty](1c4aa1ba-ad56-54d1-6e0d-2a82f7b9f4a9.md)** object.
