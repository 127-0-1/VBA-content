
# LegendEntries.Parent Property (Word)

Returns the parent for the specified object. Read-only  **Object** .


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[LegendEntries](3d130934-8a2d-a2f5-b609-3ab34f406dc4.md)** object.
