
# ShapeRange.Width Property (Word)

Returns or sets the width, in points, of the shapes within the range. Read/write  **Long** .


## Syntax

 _expression_ . **Width**

 _expression_ A variable that represents a **[ShapeRange](7112acc0-e241-16ef-77bc-101b72d05af0.md)** object.
