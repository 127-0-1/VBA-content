
# ListEntries.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **ListEntries** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[ListEntries](cfd3c706-5b69-338f-b104-e12526b89f47.md)** collection.
