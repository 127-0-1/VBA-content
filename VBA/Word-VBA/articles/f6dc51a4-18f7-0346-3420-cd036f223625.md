
# Revision.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Revision** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[Revision](e6f64467-a438-88f1-60f9-975365a1430e.md)** object.
