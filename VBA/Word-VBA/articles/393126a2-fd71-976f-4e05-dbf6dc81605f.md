
# Axis.AxisGroup Property (Word)

Returns the type of axis group. Read-only  **[XlAxisGroup](ed3ff1ce-28de-165d-bbfa-f3d770f32522.md)** .


## Syntax

 _expression_ . **AxisGroup**

 _expression_ A variable that represents an **[Axis](3a7ad7d8-d397-a79a-eb6a-a5f0822cbe5d.md)** object.
