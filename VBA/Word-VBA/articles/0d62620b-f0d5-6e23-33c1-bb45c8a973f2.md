
# TwoInitialCapsExceptions.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **TwoInitialCapsExceptions** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[TwoInitialCapsExceptions](21af2d69-8d76-026d-2002-8d69b4ab8aef.md)** collection.
