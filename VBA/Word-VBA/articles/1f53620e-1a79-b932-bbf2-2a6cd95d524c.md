
# Chart.Refresh Method (Word)

Causes the specified chart to be redrawn immediately.


## Syntax

 _expression_ . **Refresh**

 _expression_ A variable that represents a **[Chart](366a825e-0daf-dbb7-b6f2-e7ce1a5ee2ef.md)** object.
