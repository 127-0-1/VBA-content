
# MailMergeFields.Count Property (Word)

Returns a  **Long** that represents the number of mail merge fields in the collection. Read-only.


## Syntax

 _expression_ . **Count**

 _expression_ Required. A variable that represents a **[MailMergeFields](9d2dfd45-c52b-500e-15bf-1e678e6c1e92.md)** collection.
