
# FileConverter.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **FileConverter** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[FileConverter](41af2a9b-75cc-253d-4954-4fb42c88530f.md)** object.
