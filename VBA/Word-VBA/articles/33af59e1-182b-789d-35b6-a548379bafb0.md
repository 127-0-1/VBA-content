
# Breaks.Count Property (Word)

Returns the number of items in the  **Breaks** collection. Read-only **Long** .


## Syntax

 _expression_ . **Count**

 _expression_ A variable that represents a **[Breaks](c6a9213d-c4bc-a9a6-3889-a32446f1fc25.md)** object.
