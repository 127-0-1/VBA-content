
# EndnoteOptions.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **EndnoteOptions** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents an **[EndnoteOptions](b63cf439-2297-fec9-ba36-66ad3f43dcbc.md)** collection.
