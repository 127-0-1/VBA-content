
# Characters.Count Property (Word)

Returns the number of characters in the  **Characters** collection. Read-only **Long** .


## Syntax

 _expression_ . **Count**

 _expression_ A variable that represents a **[Characters](6d22ae7a-128d-134d-9136-1cdd5a8d9941.md)** collection.
