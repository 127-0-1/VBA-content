
# DataLabel.Parent Property (Word)

Returns the parent for the specified object. Read-only  **Object** .


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[DataLabel](b955596d-ac94-1e18-4e72-cdf090fc1f9e.md)** object.
