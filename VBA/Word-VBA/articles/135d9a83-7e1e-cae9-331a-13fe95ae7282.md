
# OMathGroupChar.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **OMathGroupChar** object.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[OMathGroupChar](e8f81e9d-86e7-bf52-5382-ad7d26b18af8.md)** object.
