
# DataLabel.HorizontalAlignment Property (Word)

Returns or sets the horizontal alignment for the specified object. Read/write  **Variant** .


## Syntax

 _expression_ . **HorizontalAlignment**

 _expression_ A variable that represents a **[DataLabel](b955596d-ac94-1e18-4e72-cdf090fc1f9e.md)** object.


## Remarks

You can set the value of this property to one of the following  **[XlConstants](90d5b59c-8fa6-171c-4f77-01536bc9a1d3.md)** constants:


-  **xlCenter**
    
-  **xlDistributed**
    
-  **xlJustify**
    
-  **xlLeft**
    
-  **xlRight**
    


Some of these constants may not be available to you, depending on the language support (for example, U.S. English) that you have selected ohttp:r installed.
