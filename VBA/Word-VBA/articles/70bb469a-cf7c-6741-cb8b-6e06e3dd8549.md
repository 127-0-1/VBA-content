
# AutoCorrectEntry.Delete Method (Word)

Deletes the specified AutoCorrect entry.


## Syntax

 _expression_ . **Delete**

 _expression_ Required. A variable that represents an **[AutoCorrectEntry](33173958-42eb-00ef-7f37-41f95ed47f87.md)** object.
