
# ShapeNode.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **ShapeNode** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[ShapeNode](d5afb71a-a218-57f3-87f0-171094ba6610.md)** object.
