
# OMathAutoCorrectEntries.Count Property (Word)

Returns the number of items in the  **OMathAutoCorrectEntries** collection. Read-only **Long** .


## Syntax

 _expression_ . **Count**

 _expression_ A variable that represents a **[OMathAutoCorrectEntries](3dd3bfab-3248-1832-5f86-68b3110e365b.md)** object.
