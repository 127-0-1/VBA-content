
# Rows.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Rows** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[Rows](cd83d0ef-f743-1886-54de-497017c5f542.md)** collection.
