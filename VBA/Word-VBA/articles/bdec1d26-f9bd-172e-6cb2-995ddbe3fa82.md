
# Column.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **Column** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[Column](49d68571-2a57-6795-34b9-eb09aeb43043.md)** object.
