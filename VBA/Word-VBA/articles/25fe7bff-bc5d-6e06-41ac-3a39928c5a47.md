
# ShapeRange.Shadow Property (Word)

Returns a  **ShadowFormat** object that represents the shadow formatting for the specified shape.


## Syntax

 _expression_ . **Shadow**

 _expression_ Required. A variable that represents a **[ShapeRange](7112acc0-e241-16ef-77bc-101b72d05af0.md)** object.
