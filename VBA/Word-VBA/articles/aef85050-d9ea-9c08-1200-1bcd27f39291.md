
# OMathFunctions.Count Property (Word)

Returns the number of items in the  **OMathFunctions** collection. Read-only **Long** .


## Syntax

 _expression_ . **Count**

 _expression_ A variable that represents a **[OMathFunctions](163c6af9-5cb9-93bb-f1c2-b9ebeca28f6a.md)** object.
