
# FormField.Next Property (Word)

Returns the next form field in the collection. Read-only.


## Syntax

 _expression_ . **Next**

 _expression_ A variable that represents a **[FormField](c3c07344-06b2-fe86-6fcb-b9c63a991bcc.md)** object.
