
# ChartFormat.Parent Property (Word)

Returns the parent for the specified object. Read-only  **Object** .


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **[ChartFormat](5f6546e8-c2fd-eec5-27a9-f2fd2c058f16.md)** object.
