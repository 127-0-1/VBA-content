
# AutoTextEntry.Delete Method (Word)

Deletes the specified object.


## Syntax

 _expression_ . **Delete**

 _expression_ Required. A variable that represents an **[AutoTextEntry](37a2cf05-ae07-d411-9bd8-ab4726b303a9.md)** object.
