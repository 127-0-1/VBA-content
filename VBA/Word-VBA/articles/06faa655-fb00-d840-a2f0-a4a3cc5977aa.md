
# TaskPanes.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **TaskPanes** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[TaskPanes](a560a41b-a1d7-175a-b475-af742c9fa1f8.md)** collection.
