
# OtherCorrectionsException.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **OtherCorrectionsException** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents an **[OtherCorrectionsException](f3c92186-0d3a-0585-b545-3a94e27a7d7b.md)** object.
