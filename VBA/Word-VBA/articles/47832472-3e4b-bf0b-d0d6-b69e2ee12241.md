
# StyleSheet.Parent Property (Word)

Returns an  **Object** that represents the parent object of the specified **StyleSheet** object.


## Syntax

 _expression_ . **Parent**

 _expression_ Required. A variable that represents a **[StyleSheet](5e576ff8-c458-f5bd-730d-9db827c4f76e.md)** object.
