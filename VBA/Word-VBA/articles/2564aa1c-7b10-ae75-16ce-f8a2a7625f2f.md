
# Axis.Border Property (Word)

Returns the border of the object. Read-only  **[ChartBorder](eea90670-c599-2ec8-5b7b-c946a4bcd638.md)** .


## Syntax

 _expression_ . **Border**

 _expression_ A variable that represents an **[Axis](3a7ad7d8-d397-a79a-eb6a-a5f0822cbe5d.md)** object.
