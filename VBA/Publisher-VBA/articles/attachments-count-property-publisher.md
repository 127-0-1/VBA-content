# <a name="attachmentscount-property-publisher"></a>Свойство Attachments.Count (издатель)

Возвращает количество объектов **[вложений](attachment-object-publisher.md)** в коллекции **[вложений](attachments-object-publisher.md)** объекта **[EmailMergeEnvelope](emailmergeenvelope-object-publisher.md)** . Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Count**

 переменная _expression_A, представляющий коллекцию **вложений** .


### <a name="return-value"></a>Возвращаемое значение

Длинный
