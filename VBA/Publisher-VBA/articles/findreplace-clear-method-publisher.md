# <a name="findreplaceclear-method-publisher"></a>Метод FindReplace.Clear (издатель)

Удаляет заданным критериям поиска в операции поиска и замены.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Очистить**

 переменная _expression_A, представляет собой объект- **FindReplace** .
