# <a name="printablerecttop-property-publisher"></a>Свойство PrintableRect.Top (издатель)

Возвращает **один** , представляющий расстояние (в точках) от верхнего края листа принтера по верхнему краю прямоугольника, подготовленных к печати. Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **В начало**

 переменная _expression_A, представляет собой объект- **PrintableRect** .
