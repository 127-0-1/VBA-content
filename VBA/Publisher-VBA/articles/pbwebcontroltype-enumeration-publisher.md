# <a name="pbwebcontroltype-enumeration-publisher"></a>Перечисление PbWebControlType (издатель)

Указывает тип элемента управления веб-формы для добавления.



|**Имя**|**Значение**|**Описание**|
|:-----|:-----|:-----|
| **pbWebControlCheckBox**|100|Добавление флажка.|
| **pbWebControlCommandButton**|101|Добавление кнопки команды.|
| **pbWebControlHotSpot**|110|Добавляет активной зоны.|
| **pbWebControlHTMLFragment**|107|Добавляет фрагмент HTML-кода.|
| **pbWebControlListBox**|102|Добавление поля со списком.|
| **pbWebControlMultiLineTextBox**|103|Добавляет область несколько строк текста.|
| **pbWebControlOptionButton**|104|Добавление переключателя.|
| **pbWebControlSingleLineTextBox**|105|Добавление однострочного текстового поля.|
| **pbWebControlWebComponent**|106|Добавление веб-компонент одну строку.|
