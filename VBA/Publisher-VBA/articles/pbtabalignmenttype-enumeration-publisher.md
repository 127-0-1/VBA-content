# <a name="pbtabalignmenttype-enumeration-publisher"></a>Перечисление PbTabAlignmentType (издатель)

Представляет выравнивание для заданной позиции табуляции.



|**Имя**|**Значение**|**Описание**|
|:-----|:-----|:-----|
| **pbTabAlignmentCenter**|1|Выравнивание centeral табуляции|
| **pbTabAlignmentDecimal**|3|Выравнивание Decimal табуляции|
| **pbTabAlignmentLeading**|0|Начальные выравнивание табуляции|
| **pbTabAlignmentTrailing**|2|В конце выравнивание табуляции|
