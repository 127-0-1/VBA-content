# <a name="pbselectiontype-enumeration-publisher"></a>Перечисление PbSelectionType (издатель)

Представляет тип выделения. 



|**Имя**|**Значение**|**Описание**|
|:-----|:-----|:-----|
| **pbSelectionNone**|0|Выбор введите нет.|
| **pbSelectionShape**|1|Выбор типа фигуры.|
| **pbSelectionShapeSubSelection**|4|Выбор типа Частичное выделение.|
| **pbSelectionTableCells**|3|Выбор типа ячейки.|
| **pbSelectionText**|2|Выбор типа text.|
