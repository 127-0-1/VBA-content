# <a name="sectionstartpageindex-property-publisher"></a>Свойство Section.StartPageIndex (издатель)

Возвращает номер страницы, который начинается с указанного объекта **раздела** . Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **StartPageIndex**

 переменная _expression_A, представляет собой объект **раздела** .


### <a name="return-value"></a>Возвращаемое значение

Длинный
