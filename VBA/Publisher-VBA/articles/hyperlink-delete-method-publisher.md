# <a name="hyperlinkdelete-method-publisher"></a>Метод Hyperlink.Delete (издатель)

Удаляет указанный объект.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Удаление**

 переменная _expression_A, представляющий объект **гиперссылки** .


## <a name="remarks"></a>Заметки

Если указанный объект не существует, возникает ошибка времени выполнения.
