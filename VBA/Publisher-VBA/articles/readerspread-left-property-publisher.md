# <a name="readerspreadleft-property-publisher"></a>Свойство ReaderSpread.Left (издатель)

Возвращает значение типа **одного** , указывающее положение (в точках) левого края ширина из рабочей области чтения. Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Слева**

 переменная _expression_A, представляет собой объект- **ReaderSpread** .
