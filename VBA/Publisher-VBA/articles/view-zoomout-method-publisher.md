# <a name="viewzoomout-method-publisher"></a>Метод View.ZoomOut (издатель)

Уменьшает масштаб указанное представление.


## <a name="syntax"></a>Синтаксис

 _выражение_. **ZoomOut**

 переменная _expression_A, представляющий объект **View** .


## <a name="example"></a>Пример

В этом примере уменьшает масштаб активного представления.


```vb
Sub Zoom() 
 ActiveView.ZoomOut 
End Sub
```
