# <a name="shaperangereflection-property-publisher"></a>Свойство ShapeRange.Reflection (издатель)

Возвращает объект **[ReflectionFormat](reflectionformat-object-publisher.md)** , представляющий отражения, форматирования для диапазона фигур. Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Отражения**

 переменная _expression_A, представляющий объект **ShapeRange** .


## <a name="return-value"></a>Возвращаемое значение

 **REFLECTIONFORMAT**
