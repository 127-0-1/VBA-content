# <a name="buildingblockdelete-method-publisher"></a>Метод BuildingBlock.Delete (издатель)

Удаляет указанный объект **BuildingBlock** из **[BuildingBlocks](buildingblocks-object-publisher.md)** коллекции, возвращаемой ** [Document.AvailableBuildingBlocks](document-availablebuildingblocks-property-publisher.md)** свойства для текущего документа.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Удаление**

 переменная _expression_A, представляет собой объект- **BuildingBlock** .
