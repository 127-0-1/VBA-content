# <a name="shapedelete-method-publisher"></a>Метод Shape.Delete (издатель)

Удаляет указанный объект.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Удаление**

 переменная _expression_A, представляющий объект **фигуры** .


## <a name="remarks"></a>Заметки

Если указанный объект не существует, возникает ошибка времени выполнения.

Закрытие источника данных для слияния удаляет фигуры, представляющий область данных страницы публикации, связанный с источником данных
