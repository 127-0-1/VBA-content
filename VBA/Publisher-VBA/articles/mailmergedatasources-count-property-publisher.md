# <a name="mailmergedatasourcescount-property-publisher"></a>Свойство MailMergeDataSources.Count (издатель)

Возвращает число объектов **вывода** в коллекции **MailMergeDataSources** активных документов. Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Count**

 переменная _expression_A, представляющий коллекцию **MailMergeDataSources** .


### <a name="return-value"></a>Возвращаемое значение

Длинный
