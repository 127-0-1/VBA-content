# <a name="shaperangewidth-property-publisher"></a>Свойство ShapeRange.Width (издатель)

Возвращает значение **типа Variant** , представляющее ширину (в точках) указанного диапазона фигур. Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Ширина**

 переменная _expression_A, представляющий объект **ShapeRange** .
