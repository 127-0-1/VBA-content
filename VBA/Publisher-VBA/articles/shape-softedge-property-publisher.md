# <a name="shapesoftedge-property-publisher"></a>Свойство Shape.SoftEdge (издатель)

Возвращает объект **[SoftEdgeFormat](softedgeformat-object-publisher.md)** , представляющий мягкий край форматирования для фигуры. Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Мягкий край**

 переменная _expression_A, представляющий объект **фигуры** .


## <a name="return-value"></a>Возвращаемое значение

 **SOFTEDGEFORMAT**
