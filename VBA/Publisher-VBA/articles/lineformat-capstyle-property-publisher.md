# <a name="lineformatcapstyle-property-publisher"></a>Свойство LineFormat.CapStyle (издатель)

Задает стиль окончания строк, а также формат пунктирной линии. Режим чтения и записи.


## <a name="syntax"></a>Синтаксис

 _выражение_. **CapStyle**

 переменная _expression_A, представляет собой объект- **LineFormat** .


## <a name="return-value"></a>Возвращаемое значение

 **MSOLINECAPSTYLE**
