# <a name="fillformatgradientangle-property-publisher"></a>Свойство FillFormat.GradientAngle (издатель)

Возвращает или задает угол градиентной заливки для заливки указанного формата. Чтение и запись.


## <a name="syntax"></a>Синтаксис

 _выражение_. **GradientAngle**

 переменная _expression_A, представляет собой объект- **FillFormat** .


## <a name="return-value"></a>Возвращаемое значение

 **Один**


## <a name="remarks"></a>Заметки

Можно указать градиентной заливки в форматирования для различных форм. Диапазон допустимых значений для свойства **GradientAngle** — от 0 359.9.
