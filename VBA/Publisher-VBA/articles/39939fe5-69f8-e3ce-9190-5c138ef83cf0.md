
# LineFormat.JoinStyle Property (Publisher)

Returns of sets the join style for the specified line. Read-write.


## Syntax

 _expression_. **JoinStyle**

 _expression_A variable that represents a  **LineFormat** object.


## Return value

 **MSOLINEJOINSTYLE**
