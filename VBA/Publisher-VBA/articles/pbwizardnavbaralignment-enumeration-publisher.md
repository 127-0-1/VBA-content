# <a name="pbwizardnavbaralignment-enumeration-publisher"></a>Перечисление PbWizardNavBarAlignment (издатель)

Представляет горизонтальное выравнивание кнопок в наборе панель навигации Web.



|**Имя**|**Значение**|**Описание**|
|:-----|:-----|:-----|
| **pbnbAlignCenter**|2|Центр по правому краю|
| **pbnbAlignLeft**|1|По левому краю|
| **pbnbAlignRight**|3|По правому краю|
