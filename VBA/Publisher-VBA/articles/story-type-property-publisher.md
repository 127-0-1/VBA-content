# <a name="storytype-property-publisher"></a>Свойство Story.Type (издатель)

Указывает тип материала. Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Тип**

 переменная _expression_A, представляет собой объект- **материала** .


## <a name="remarks"></a>Заметки

Значение свойства **типа** может иметь одно из следующих констант **PbStoryType**



| **pbStoryContinuedFrom**|| **pbStoryContinuedOn**|| **pbStoryTable**|| **pbStoryTextFrame**|
