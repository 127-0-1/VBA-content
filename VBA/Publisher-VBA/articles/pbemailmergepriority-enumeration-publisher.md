# <a name="pbemailmergepriority-enumeration-publisher"></a>pbEmailMergePriority перечисления (издатель)

Константы для составных электронных сообщений приоритет обмене с ** [EmailMergeEnvelope.Priority](emailmergeenvelope-priority-property-publisher.md)** свойство.



|**Имя**|**Значение**|**Описание**|
|:-----|:-----|:-----|
| **pbPriorityHigh**|1|Высокий приоритет|
| **pbPriorityLow**|2|Низкая важность|
| **pbPriorityNone**|0|Не установлен приоритет|
