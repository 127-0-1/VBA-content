# <a name="pbsubmitdataformattype-enumeration-publisher"></a>Перечисление PbSubmitDataFormatType (издатель)

Представляет формат для использования при сохранении в файл данных веб-форм. 



|**Имя**|**Значение**|**Описание**|
|:-----|:-----|:-----|
| **pbSubmitDataFormatCSV**|3|Сохранение данных веб-форм в файле с разделителями-запятыми.|
| **pbSubmitDataFormatHTML**|1|Сохранение данных веб-форм в HTML-файл.|
| **pbSubmitDataFormatRichText**|2|Сохранение данных веб-форм форматированный файл.|
| **pbSubmitDataFormatTab**|4|Сохранение данных веб-форм в разделители — знаки табуляции текстовый файл.|
