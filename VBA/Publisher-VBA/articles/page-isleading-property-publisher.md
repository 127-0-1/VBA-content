# <a name="pageisleading-property-publisher"></a>Свойство Page.IsLeading (издатель)

 **Значение true,** Если указанный объект **Page** является основную страницу двух страницах. Только для чтения **типа Boolean**.


## <a name="syntax"></a>Синтаксис

 _выражение_. **IsLeading**

 переменная _expression_A, представляющий объект **страницы** .


### <a name="return-value"></a>Возвращаемое значение

Логический


## <a name="example"></a>Пример

Следующий пример отображает для каждой страницы ли страница находится в конце строки или основную страницу в публикации.
