# <a name="selectionunselect-method-publisher"></a>Метод Selection.Unselect (издатель)

Отмена текущего выделения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Снятие выделения**

 переменная _expression_A, представляющий объект **Selection** .


## <a name="example"></a>Пример

В этом примере показано, как отменить текущего выделения в активной публикации.


```vb
ActiveDocument.Selection.Unselect
```
