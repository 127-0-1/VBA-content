# <a name="shaperangesaveasbuildingblock-method-publisher"></a>Метод ShapeRange.SaveAsBuildingBlock (издатель)

Сохраняет диапазон одну форму в виде стандартного блока. Возвращает итоговый объект **[BuildingBlock](buildingblock-object-publisher.md)** .


## <a name="syntax"></a>Синтаксис

 _выражение_. **SaveAsBuildingBlock** ( **_Имя_**)

 переменная _expression_A, представляющий объект **ShapeRange** .


### <a name="parameters"></a>Параметры



|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
|Имя|Обязательное свойство.| **String**|Имя нового стандартного блока, который требуется создать.|
