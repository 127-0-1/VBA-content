# <a name="pbsaveoptions-enumeration-publisher"></a>Перечисление PbSaveOptions (издатель)

Указывает, что следует сделать Microsoft Publisher Если уже открытой публикации с несохраненными изменениями.



|**Имя**|**Значение**|**Описание**|
|:-----|:-----|:-----|
| **pbDoNotSaveChanges**|3|Закройте открыть публикацию без сохранения изменений.|
| **pbPromptToSaveChanges**|1|По умолчанию. Запрашивать у пользователя, следует ли сохранить изменения в открытой публикации.|
| **pbSaveChanges**|2|Сохраните открыть публикацию перед закрытием.|
