Возвращает доступные элементы коллекции **[BuildingBlocks](buildingblocks-object-publisher.md)** текущего документа. Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **AvailableBuildingBlocks**

 переменная _expression_A, представляющий объект **Document** .


### <a name="return-value"></a>Возвращаемое значение

 **BuildingBlocks**
