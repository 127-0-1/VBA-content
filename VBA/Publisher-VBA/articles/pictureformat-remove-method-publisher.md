# <a name="pictureformatremove-method-publisher"></a>Метод PictureFormat.Remove (издатель)

Удаляет изображение из фрейма, отправляемых из него местозаполнитель изображения. Все предыдущие форматирование, применяемое к рисунок сохраняется. Возвращает значение nothing.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Удаление**

 переменная _expression_A, представляет собой объект- [PictureFormat](pictureformat-object-publisher.md) .
