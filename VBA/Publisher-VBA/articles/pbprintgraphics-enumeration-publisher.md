# <a name="pbprintgraphics-enumeration-publisher"></a>Перечисление PbPrintGraphics (издатель)

Представляет решение, на которой можно вставленные изображения на печать в указанной публикации.



|**Имя**|**Значение**|**Описание**|
|:-----|:-----|:-----|
| **pbPrintHighResolution**|1|По умолчанию. Печать связанных рисунков с помощью версии связанного высоким разрешением.|
| **pbPrintLowResolution**|2|Печать связанных рисунков с помощью версии с низким разрешением заполнитель, которая хранится в публикации.|
| **pbPrintNoGraphics**|3|Печать поле вместо связанных рисунков.|
