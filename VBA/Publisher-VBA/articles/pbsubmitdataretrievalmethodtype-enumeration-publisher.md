# <a name="pbsubmitdataretrievalmethodtype-enumeration-publisher"></a>Перечисление PbSubmitDataRetrievalMethodType (издатель)

Представляет способ обработки данных из веб-форму. 



|**Имя**|**Значение**|**Описание**|
|:-----|:-----|:-----|
| **pbSubmitDataRetrievalEmail**|2|Процессы данные формы, отправив сообщение электронной почты на указанный адрес электронной почты.|
| **pbSubmitDataRetrievalProgram**|3|Процессы форм данных с помощью программы скрипт, предоставляемые поставщиком услуг Интернета.|
| **pbSubmitDataRetrievalSaveOnServer**|1|Сохранение данных формы для файлов, хранящихся на веб-сервере.|
