# <a name="shaperangeglow-property-publisher"></a>Свойство ShapeRange.Glow (издатель)

Возвращает объект **[GlowFormat](glowformat-object-publisher.md)** , представляющий свечения форматирования для диапазона фигур. Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Свечения**

 переменная _expression_A, представляющий объект **ShapeRange** .


## <a name="return-value"></a>Возвращаемое значение

 **GLOWFORMAT**
