# <a name="lineformatgradientstyle-property-publisher"></a>Свойство LineFormat.GradientStyle (издатель)

Возвращает тип градиента для указанной строки. Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **GradientStyle**

 переменная _expression_A, представляет собой объект- **LineFormat** .


## <a name="return-value"></a>Возвращаемое значение

 **MsoGradientStyle**


## <a name="remarks"></a>Заметки

Попытка получить значение этого свойства для строки, не имеющим градиент приводит к ошибке. Свойство **[типа](lineformat-type-property-publisher.md)** определить, имеет ли строка градиент.
