# <a name="pbrecipientlistfiletype-enumeration-publisher"></a>Перечисление PbRecipientListFileType (издатель)

Константы передается ** [MailMerge.ExportRecipientList](mailmerge-exportrecipientlist-method-publisher.md)** метод, укажите тип файла для сохранения экспортированного слияния почты списка получателей.



|**Имя**|**Значение**|**Описание**|
|:-----|:-----|:-----|
| **pbAsCsvFile**|1|Сохраните как CSV-файла с разделителями запятыми.|
| **pbAsMdbFile**|0|Сохраните как Microsoft Office Access MDB-файлу.|
