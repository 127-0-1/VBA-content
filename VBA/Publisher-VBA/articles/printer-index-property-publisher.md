# <a name="printerindex-property-publisher"></a>Свойство Printer.Index (издатель)

Возвращает значение типа **Long** , представляющее номер конкретного принтера в коллекции указанного принтера.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Индекс**

 переменная _expression_A, представляющий объект **Printer** .


### <a name="return-value"></a>Возвращаемое значение

Длинный
