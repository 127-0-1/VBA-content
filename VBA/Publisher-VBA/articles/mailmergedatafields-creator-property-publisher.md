# <a name="mailmergedatafieldscreator-property-publisher"></a>Свойство MailMergeDataFields.Creator (издатель)

Возвращает значение типа **Long** , который представляет приложение, в котором был создан указанный объект. Например если объект был создан в Microsoft Publisher, данное свойство возвращает шестнадцатеричное число 4D 505542, который представляет строку «MSPB». Это значение также могут быть представлены константу.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Создатель**

 переменная _expression_A, представляет собой объект- **MailMergeDataFields** .
