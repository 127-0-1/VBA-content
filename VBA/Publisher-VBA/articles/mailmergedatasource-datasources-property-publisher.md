# <a name="mailmergedatasourcedatasources-property-publisher"></a>Свойство MailMergeDataSource.DataSources (издатель)

Возвращает коллекцию **MailMergeDataSources** , которая включает в себя родительский объект **вывода** . Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Источники данных**

 переменная _expression_A, представляющий объект **вывода** .


### <a name="return-value"></a>Возвращаемое значение

MailMergeDataSources
