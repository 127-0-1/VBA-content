# <a name="shaperangeleft-property-publisher"></a>Свойство ShapeRange.Left (издатель)

Возвращает значение **типа Variant** , указывающее расстояние от левого края страницы до левого края всех фигур в диапазон указанной фигуры. Числовые значения находятся в точках; все остальные значения находятся в любой измерения, поддерживаемых Publisher (например, «2,5 дюйма»). Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Слева**

 переменная _expression_A, представляющий объект **ShapeRange** .
