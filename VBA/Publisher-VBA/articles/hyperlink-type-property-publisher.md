# <a name="hyperlinktype-property-publisher"></a>Свойство Hyperlink.Type (издатель)

Возвращает константу **MsoHyperlinkType** , представляющий тип гиперссылки. Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Тип**

 переменная _expression_A, представляющий объект **гиперссылки** .


## <a name="remarks"></a>Заметки

Значение свойства **типа** может иметь одно из следующих констант **MsoHyperlinkType** .



| **msoHyperlinkInlineShape**|| **msoHyperlinkRange**|| **msoHyperlinkShape**|
