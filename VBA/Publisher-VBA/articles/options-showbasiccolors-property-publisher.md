# <a name="optionsshowbasiccolors-property-publisher"></a>Свойство Options.ShowBasicColors (издатель)

Возвращает или задает значение **Boolean** , указывающее, отображает ли Microsoft Publisher основные цвета в цветовой палитры; **Значение true,** чтобы показывать основные цвета в палитре. Чтение и запись.


## <a name="syntax"></a>Синтаксис

 _выражение_. **ShowBasicColors**

 переменная _expression_A, представляет собой объект- **Параметры** .


### <a name="return-value"></a>Возвращаемое значение

Логический


## <a name="example"></a>Пример

В следующем примере задается Publisher не показывать основные цвета в цветовой палитры.


```vb
Options.ShowBasicColors = False
```
