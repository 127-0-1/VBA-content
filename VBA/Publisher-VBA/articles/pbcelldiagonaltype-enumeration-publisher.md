# <a name="pbcelldiagonaltype-enumeration-publisher"></a>Перечисление PbCellDiagonalType (издатель)

Представляет ячейку, в которой диагонали разделение.



|**Имя**|**Значение**|**Описание**|
|:-----|:-----|:-----|
| **pbTableCellDiagonalDown**|2|Диагональ вниз|
| **pbTableCellDiagonalMixed**|-2|Mixed вправо|
| **pbTableCellDiagonalNone**|0|Не разбиение по диагонали|
| **pbTableCellDiagonalUp**|1|Диагональ вверх|
