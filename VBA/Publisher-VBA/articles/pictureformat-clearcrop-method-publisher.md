# <a name="pictureformatclearcrop-method-publisher"></a>Метод PictureFormat.ClearCrop (издатель)

Удаляет фигуры рисунок из изображения и сбрасывает области обрезки в степени исходного изображения перед обрезки. Возвращает значение nothing.


## <a name="syntax"></a>Синтаксис

 _выражение_. **ClearCrop**

 переменная _expression_A, представляет собой объект- [PictureFormat](pictureformat-object-publisher.md) .
