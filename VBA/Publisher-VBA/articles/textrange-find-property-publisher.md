# <a name="textrangefind-property-publisher"></a>Свойство TextRange.Find (издатель)

Возвращает объект **FindReplace** из указанного объекта **TextRange** . Объект **FindReplace** используется для выполнения поиска и замены в диапазоне указанный текст.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Поиск**

 переменная _expression_A, представляющий объект **TextRange** .
