# <a name="pictureformatrecoloredpicturecolor-property-publisher"></a>Свойство PictureFormat.RecoloredPictureColor (издатель)

Возвращает цвет, которая была применена к нужному изображению, представленного объектом **PictureFormat** родительского. Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **RecoloredPictureColor**

 переменная _expression_A, представляет собой объект- **PictureFormat** .


### <a name="return-value"></a>Возвращаемое значение

ColorFormat
