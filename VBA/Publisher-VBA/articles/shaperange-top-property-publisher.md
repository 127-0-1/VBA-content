# <a name="shaperangetop-property-publisher"></a>Свойство ShapeRange.Top (издатель)

Возвращает значение **типа Variant** , представляющее расстояние между верхней части страницы и верхней фигуры в диапазоне фигур. Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **В начало**

 переменная _expression_A, представляющий объект **ShapeRange** .
