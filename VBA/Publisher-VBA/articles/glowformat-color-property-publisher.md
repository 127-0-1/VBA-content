# <a name="glowformatcolor-property-publisher"></a>Свойство GlowFormat.Color (издатель)

Возвращает объект **[ColorFormat](colorformat-object-publisher.md)** , который представляет цвет для эффект свечения. Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Цвет**

 переменная _expression_A, представляет собой объект- **GlowFormat** .


## <a name="return-value"></a>Возвращаемое значение

 **COLORFORMAT**
