# <a name="pbmailmergedatafieldtype-enumeration-publisher"></a>Перечисление PbMailMergeDataFieldType (издатель)

Представляет тип данных, содержащихся в поле данных.



|**Имя**|**Значение**|**Описание**|
|:-----|:-----|:-----|
| **pbMailMergeDataFieldPicture**|1|Содержит изображения.|
| **pbMailMergeDataFieldString**|0|Содержит строку.|
