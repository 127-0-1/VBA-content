# <a name="mailmergedatasourceismaster-property-publisher"></a>Свойство MailMergeDataSource.IsMaster (издатель)

Указывает, является ли родительский объект **вывода** основного источника данных (сочетание все источники данных, подключенные к текущей публикации). Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **IsMaster**

 переменная _expression_A, представляющий объект **вывода** .

Логический
