# <a name="pbparagraphalignmenttype-enumeration-publisher"></a>Перечисление PbParagraphAlignmentType (издатель)

 Представляет выравнивание для указанного абзацев.



|**Имя**|**Значение**|**Описание**|
|:-----|:-----|:-----|
| **pbParagraphAlignmentCenter**|1|Выравнивание по центру|
| **pbParagraphAlignmentDistribute**|4|Распространение выравнивание|
| **pbParagraphAlignmentDistributeAll**|9|Распространение всех|
| **pbParagraphAlignmentDistributeCenterLast**|10|Распространение центр последнего|
| **pbParagraphAlignmentDistributeEastAsia**|5|Распространение восточно-азиатские|
| **pbParagraphAlignmentInterCluster**|8|Взаимодействия между каналами кластера|
| **pbParagraphAlignmentInterIdeograph**|7|Взаимодействия между каналами алфавита|
| **pbParagraphAlignmentInterWord**|3|Взаимодействия между каналами Word|
| **pbParagraphAlignmentJustified**|6|По ширине|
| **pbParagraphAlignmentKashida**|11|Кашиды|
| **pbParagraphAlignmentLeft**|0|Выравнивание слева|
| **pbParagraphAlignmentMixed**|-9999999|Смешанные выравнивание|
| **pbParagraphAlignmentRight**|2|Выравнивание по правому краю|
