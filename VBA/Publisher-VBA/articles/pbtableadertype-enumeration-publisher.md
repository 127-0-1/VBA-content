# <a name="pbtableadertype-enumeration-publisher"></a>Перечисление PbTabLeaderType (издатель)

Представляет заполнитель для позиции табуляции. 



|**Имя**|**Значение**|**Описание**|
|:-----|:-----|:-----|
| **pbTabLeaderBullet**|5|Вкладка маркера руководитель|
| **pbTabLeaderDashes**|2|Вкладка ведущий тире|
| **pbTabLeaderDot**|1|Ведущий dot вкладки|
| **pbTabLeaderLine**|3|Вкладка ведущий строки|
| **pbTabLeaderNone**|0|Заполнитель нет|
