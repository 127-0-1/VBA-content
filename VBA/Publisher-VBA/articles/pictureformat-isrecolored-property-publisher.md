# <a name="pictureformatisrecolored-property-publisher"></a>Свойство PictureFormat.IsRecolored (издатель)

Возвращает **значение True,** Если изображение, представленное родительский объект **PictureFormat** были перекрашены, в пользовательском интерфейсе или с помощью ** [PictureFormat.Recolor](pictureformat-recolor-method-publisher.md)** метод. Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **IsRecolored**

 переменная _expression_A, представляет собой объект- **PictureFormat** .


### <a name="return-value"></a>Возвращаемое значение

MsoTriState
