# <a name="textrangelinescount-property-publisher"></a>Свойство TextRange.LinesCount (издатель)

Возвращает количество строк текста в диапазон текста, представленного объектом **TextRange** родительского. Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **LinesCount**

 переменная _expression_A, представляющий объект **TextRange** .


### <a name="return-value"></a>Возвращаемое значение

Длинный
