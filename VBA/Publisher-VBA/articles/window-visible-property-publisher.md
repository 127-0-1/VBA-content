# <a name="windowvisible-property-publisher"></a>Свойство Window.Visible (издатель)

 **Значение true,** Если окно является видимым. Чтение и запись **типа Boolean**.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Visible**

 переменная _expression_A, представляющий объект **Window** .


## <a name="example"></a>Пример

В этом примере скрывает окно Publisher.


```vb
ActiveWindow.Visible = False
```
