# <a name="pbunittype-enumeration-publisher"></a>Перечисление PbUnitType (издатель)

Представляет единицы измерения standard для Microsoft Publisher. 



|**Имя**|**Значение**|**Описание**|
|:-----|:-----|:-----|
| **pbUnitCM**|1|Задает единицы измерения см.|
| **pbUnitEmu**|4|Задает единицу измерения для ЕС.|
| **pbUnitFeet**|6|Задает единицы измерения футов.|
| **pbUnitHa**|9|Задает единицу измерения для высокой доступности.|
| **pbUnitInch**|0|Задает единицы измерения см.|
| **pbUnitKyu**|8|Задает единицу измерения для Kyu.|
| **pbUnitMeter**|7|Задает единицы измерения м.|
| **pbUnitPica**|2|Задает единицы измерения пики.|
| **pbUnitPixel**|10|Назначает единицы измерения.|
| **pbUnitPoint**|3|Задает единицы измерения в точках.|
| **pbUnitTwip**|5|Задает единицы измерения твип.|
