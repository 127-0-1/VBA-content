# <a name="pbhelptype-enumeration-publisher"></a>Перечисление PbHelpType (издатель)

Тип справки для отображения.



|**Имя**|**Значение**|**Описание**|
|:-----|:-----|:-----|
| **pbHelp**|1|Отображает диалоговое окно **справки** .|
| **pbHelpActiveWindow**|2|Отображение справки, описывающие команды, связанные с активным представлением или области.|
| **pbHelpPSSHelp**|3|Отображает сведения о поддержке продукта.|
