# <a name="phoneticguideclear-method-publisher"></a>Метод PhoneticGuide.Clear (издатель)

Удаляет фонетическую информацию из текста на японском языке.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Очистить**

 переменная _expression_A, представляет собой объект- **PhoneticGuide** .
