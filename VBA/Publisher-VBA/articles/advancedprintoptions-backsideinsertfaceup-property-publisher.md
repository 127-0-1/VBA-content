# <a name="advancedprintoptionsbacksideinsertfaceup-property-publisher"></a>Свойство AdvancedPrintOptions.BackSideInsertFaceUp (издатель)

Получает или задает ли подача принтера ручной подачи конвертов должен быть вставленный наряду сначала и лицевой стороной вверх. Чтение и запись.


## <a name="syntax"></a>Синтаксис

 _выражение_. **BackSideInsertFaceUp**

 переменная _expression_A, представляющий объект **AdvancedPrintOptions** .


### <a name="return-value"></a>Возвращаемое значение

Логический


## <a name="remarks"></a>Заметки

Настройка свойства **BackSideInsertFaceUp** соответствует ли выбран параметр **лицевой стороной вверх** или **лицевой стороной вниз** , при выборе **часовой стрелки** в разделе **способ подачи конвертов** в диалоговом окне **Настройка конверта** в интерфейсе пользователя Microsoft Publisher. (В меню **файл** выберите пункт **Настройка печати**. На вкладке **Сведения о** нажмите кнопку **Дополнительные параметры принтера**. На вкладке **Мастера установки принтера** выберите **Окно настройки конвертов**).
