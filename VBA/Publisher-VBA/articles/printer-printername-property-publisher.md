# <a name="printerprintername-property-publisher"></a>Свойство Printer.PrinterName (издатель)

Возвращает имя принтера, представленный родительский объект **Printer** . Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Имя_принтера**

 переменная _expression_A, представляющий объект **Printer** .


### <a name="return-value"></a>Возвращаемое значение

String


## <a name="remarks"></a>Заметки

Также можно использовать свойство **[InstalledPrinters](installedprinters-object-publisher.md)** Чтобы получить список имен принтеров, установленных на компьютере.
