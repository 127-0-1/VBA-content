# <a name="pblinespacingrule-enumeration-publisher"></a>Перечисление PbLineSpacingRule (издатель)

Представляет междустрочным интервалом для указанного абзацев.



|**Имя**|**Значение**|**Описание**|
|:-----|:-----|:-----|
| **pbLineSpacing1pt5**|1|Наборы абзацами строки на строку и с половиной.|
| **pbLineSpacingDouble**|2|Наборы абзацами строки для двух строк.|
| **pbLineSpacingExactly**|4|Наборы абзацами линии точно значение свойства **LineSpacing** даже в том случае, если размер шрифта используется внутри абзаца.|
| **pbLineSpacingMixed**|-9999999|Возвращаемое значение для абзаца, который имеет междустрочным интервалом разные значения.|
| **pbLineSpacingMultiple**|5|Значение свойства **LineSpacing** должен быть указан на количество строк.|
| **pbLineSpacingSingle**|0|Наборы абзацами строки для одного пространства.|
