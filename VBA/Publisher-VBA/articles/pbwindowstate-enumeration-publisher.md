# <a name="pbwindowstate-enumeration-publisher"></a>Перечисление PbWindowState (издатель)

Указывает состояние окна, Microsoft Publisher. 



|**Имя**|**Значение**|**Описание**|
|:-----|:-----|:-----|
| **pbWindowStateMaximize**|0|Окно развернуто.|
| **pbWindowStateMinimize**|1|Окно свернуто.|
| **pbWindowStateNormal**|2|Окно не развернуто и не свернуто.|
