# <a name="textrangeparagraphscount-property-publisher"></a>Свойство TextRange.ParagraphsCount (издатель)

Возвращает число абзацев текста в диапазон текста, представленного объектом **TextRange** родительского. Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **ParagraphsCount**

 переменная _expression_A, представляющий объект **TextRange** .


### <a name="return-value"></a>Возвращаемое значение

Длинный
