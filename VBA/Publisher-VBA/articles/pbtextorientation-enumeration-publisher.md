# <a name="pbtextorientation-enumeration-publisher"></a>Перечисление PbTextOrientation (издатель)

Представляет потоки текст в текстовом поле.



|**Имя**|**Значение**|**Описание**|
|:-----|:-----|:-----|
| **pbTextOrientationHorizontal**|1|Ориентация текста по горизонтали|
| **pbTextOrientationMixed**|-2|Ориентация смешанных текста|
| **pbTextOrientationRightToLeft**|256|Ориентация текста RightToLeft|
| **pbTextOrientationVerticalEastAsia**|2|Ориентация текста VerticalEastAsia|
