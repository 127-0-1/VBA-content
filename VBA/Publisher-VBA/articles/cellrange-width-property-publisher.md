# <a name="cellrangewidth-property-publisher"></a>Свойство CellRange.Width (издатель)

Возвращает значение типа **Long** , представляющих ширину (в ячейках) в таблице, диапазон ячеек или страницы. Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Ширина**

 переменная _expression_A, представляет собой объект- **CellRange** .
