# <a name="sectiondelete-method-publisher"></a>Метод Section.Delete (издатель)

Удаляет указанный объект.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Удаление**

 переменная _expression_A, представляет собой объект **раздела** .


## <a name="remarks"></a>Заметки

Если указанный объект не существует, возникает ошибка времени выполнения.
