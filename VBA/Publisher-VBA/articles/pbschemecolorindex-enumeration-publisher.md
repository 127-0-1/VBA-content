# <a name="pbschemecolorindex-enumeration-publisher"></a>Перечисление PbSchemeColorIndex (издатель)

Представляет указанный цвет текущего цветовая схема.



|**Имя**|**Значение**|**Описание**|
|:-----|:-----|:-----|
| **pbSchemeColorAccent1**|2|Задает цвет для цвет Акцент 1.|
| **pbSchemeColorAccent2**|3|Задает цвет для Accent2 цвет.|
| **pbSchemeColorAccent3**|4|Задает цвет для Accent3 цвет.|
| **pbSchemeColorAccent4**|5|Задает цвет для Accent4 цвет.|
| **pbSchemeColorAccent5**|8|Задает цвет для Accent5 цвет.|
| **pbSchemeColorFollowedHyperlink**|7|Задает цвет для последующей гиперссылкой.|
| **pbSchemeColorHyperlink**|6|Задает цветовая схема цвет гиперссылки.|
| **pbSchemeColorMain**|1|Задает цвет для main.|
| **pbSchemeColorNone**|0|Задает цветовая схема значение none.|
