# <a name="fontthreed-property-publisher"></a>Свойство Font.ThreeD (издатель)

Возвращает объект [ThreeDFormat](threedformat-object-publisher.md) , который содержит объема форматирования свойства для выбранного шрифта. Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **ThreeD**

 переменная _expression_A, представляющий объект **Font** .


## <a name="return-value"></a>Возвращаемое значение

 **THREEDFORMAT**
