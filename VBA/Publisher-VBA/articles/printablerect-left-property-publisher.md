# <a name="printablerectleft-property-publisher"></a>Свойство PrintableRect.Left (издатель)

Возвращает значение типа **одного** , указывающее расстояние (в точках) от левого края страницы принтера по левому краю прямоугольника, подготовленных к печати. Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Слева**

 переменная _expression_A, представляет собой объект- **PrintableRect** .
