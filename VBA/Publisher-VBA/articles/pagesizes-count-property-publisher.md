# <a name="pagesizescount-property-publisher"></a>Свойство PageSizes.Count (издатель)

Возвращает число объектов **PageSize** в коллекции **PageSizes** активных документов. Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Count**

 переменная _expression_A, представляющий коллекцию **PageSizes** .


### <a name="return-value"></a>Возвращаемое значение

Длинный
