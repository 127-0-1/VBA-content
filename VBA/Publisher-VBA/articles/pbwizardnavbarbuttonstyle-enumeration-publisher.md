# <a name="pbwizardnavbarbuttonstyle-enumeration-publisher"></a>Перечисление PbWizardNavBarButtonStyle (издатель)

Представляет стиль кнопок панели навигации: большой, небольшой или только текст.



|**Имя**|**Значение**|**Описание**|
|:-----|:-----|:-----|
| **pbnbButtonStyleLarge**|2|Крупные кнопки|
| **pbnbButtonStyleSmall**|1|Небольшие кнопки|
| **pbnbButtonStyleText**|3|Только для текста кнопок|
