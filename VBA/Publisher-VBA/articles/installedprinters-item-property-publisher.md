# <a name="installedprintersitem-property-publisher"></a>Свойство InstalledPrinters.Item (издатель)

Возвращает объект **[Printer](printer-object-publisher.md)** с указанным именем или по указанному индексу в коллекции **[InstalledPrinters](installedprinters-object-publisher.md)** . Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Элемент** ( **_Индекс_**)

 переменная _expression_A, представляющий коллекцию **InstalledPrinters** .


### <a name="parameters"></a>Параметры



|**Имя**|**Обязательный или необязательный**|**Тип данных**|**Описание**|
|:-----|:-----|:-----|:-----|
|Индекс|Обязательное свойство.| **Variant**|Положение имени или индекса установленного принтера.|

### <a name="return-value"></a>Возвращаемое значение

Принтера
