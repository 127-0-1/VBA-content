# <a name="pictureformatleaveblackasblack-property-publisher"></a>Свойство PictureFormat.LeaveBlackAsBlack (издатель)

Возвращает **значение True** , если при Перекрашивание изображение, представленное родительский объект **PictureFormat** , черные частей исходного изображения должен остаться черные. Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **LeaveBlackAsBlack**

 переменная _expression_A, представляет собой объект- **PictureFormat** .


### <a name="return-value"></a>Возвращаемое значение

MsoTriState


## <a name="remarks"></a>Заметки

Свойство **LeaveBlackAsBlack** соответствует параметру **Оставить черные области черные** в диалоговом окне **Перекрашивание рисунков** в интерфейсе пользователя Microsoft Publisher. (В меню **Формат** выберите пункт **изображение**и нажмите кнопку **изменить цвет**).
