# <a name="fieldtype-property-publisher"></a>Свойство Field.Type (издатель)

Указывает тип поля. Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Тип**

 переменная _expression_A, представляющий объект **поля** .


## <a name="remarks"></a>Заметки

Значение свойства **типа** может иметь одно из **[PbFieldType](pbfieldtype-enumeration-publisher.md)** константы в библиотеке типов, Microsoft Publisher.


## <a name="property-value"></a>Значение свойства

 **PBFIELDTYPE**
