# <a name="printerpapersource-property-publisher"></a>Свойство Printer.PaperSource (издатель)

Возвращает источник бумаги найдена на вкладке **Параметры публикации и бумаги** в диалоговом окне **Настройка печати** в интерфейсе пользователя Microsoft Publisher (меню " **файл** "). Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **PaperSource**

 переменная _expression_A, представляющий объект **Printer** .


### <a name="return-value"></a>Возвращаемое значение

String
