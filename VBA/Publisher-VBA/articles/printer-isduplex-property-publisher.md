# <a name="printerisduplex-property-publisher"></a>Свойство Printer.IsDuplex (издатель)

Возвращает **значение True,** Если устройство, представленная родительский объект **Printer** — это режим двусторонней печати. Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **IsDuplex**

 переменная _expression_A, представляющий объект **Printer** .
