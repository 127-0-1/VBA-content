# <a name="pbphoneticguidealignmenttype-enumeration-publisher"></a>Перечисление PbPhoneticGuideAlignmentType (издатель)

Константы передается ** [Fields.AddPhoneticGuide](fields-addphoneticguide-method-publisher.md)** метод, который определяет выравнивание фонетическое добавлен текст.



|**Имя**|**Значение**|**Описание**|
|:-----|:-----|:-----|
| **pbPhoneticGuideAlignmentCenter**|3|Выровненный по центру|
| **pbPhoneticGuideAlignmentDefault**|0|Выравнивание по умолчанию|
| **pbPhoneticGuideAlignmentLeft**|4|Выравнивание по левому краю|
| **pbPhoneticGuideAlignmentOneTwoOne**|2|Один два выравнивания|
| **pbPhoneticGuideAlignmentRight**|5|Выравнивание по правому краю|
| **pbPhoneticGuideAlignmentZeroOneZero**|1|Выравнивание ноль один нуля|
