# <a name="lineformattransparency-property-publisher"></a>Свойство LineFormat.Transparency (издатель)

Возвращает или задает степень прозрачности строк. Чтение и запись **одного**.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Прозрачность**

 переменная _expression_A, представляет собой объект- **LineFormat** .


## <a name="return-value"></a>Возвращаемое значение

 **Один**


## <a name="remarks"></a>Заметки

Значение этого свойства может быть выражены значение между 0.0 (Непрозрачный) и 1.0 (Очистить). Это свойство влияет на внешний вид строки сплошного только; он не влияет на внешний вид Узорные линии.
