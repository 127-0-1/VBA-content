# <a name="installedprinterscount-property-publisher"></a>Свойство InstalledPrinters.Count (издатель)

Возвращает число принтеров, установленных на вашем компьютере. Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Count**

 переменная _expression_A, представляющий коллекцию **InstalledPrinters** .


### <a name="return-value"></a>Возвращаемое значение

Int
