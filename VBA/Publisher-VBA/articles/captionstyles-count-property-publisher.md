# <a name="captionstylescount-property-publisher"></a>Свойство CaptionStyles.Count (издатель)

Возвращает значение типа **Long** , представляющее количество элементов в указанном семействе сайтов. Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Count**

 переменная _expression_A, представляет собой объект- **CaptionStyles** .
