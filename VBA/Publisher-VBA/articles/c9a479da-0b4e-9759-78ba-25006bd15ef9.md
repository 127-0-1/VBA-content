
# ShapeRange.Glow Property (Publisher)

Returns a  **[GlowFormat](62fd5b2a-f199-588e-c15e-ea27c5b59699.md)** object that represents the glow formatting for a range of shapes. Read-only.


## Syntax

 _expression_. **Glow**

 _expression_A variable that represents a  **ShapeRange** object.


## Return value

 **GLOWFORMAT**
