# <a name="pbstorytype-enumeration-publisher"></a>Перечисление PbStoryType (издатель)

Представляет тип материала. 



|**Имя**|**Значение**|**Описание**|
|:-----|:-----|:-----|
| **pbStoryContinuedFrom**|2|Продолжение статьи из рамка текста.|
| **pbStoryContinuedOn**|3|Вход в систему рамка текста продолжение статьи.|
| **pbStoryTable**|0|Таблица материала.|
| **pbStoryTextFrame**|1|История рамку.|
