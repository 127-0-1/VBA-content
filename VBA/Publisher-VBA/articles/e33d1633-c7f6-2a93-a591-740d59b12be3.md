
# ShadowFormat.Blur Property (Publisher)

Returns or sets a  **Single** that represents the blur level for a shadow format. Read/write.


## Syntax

 _expression_. **Blur**

 _expression_A variable that represents a  **ShadowFormat** object.


## Return value

 **Single**
