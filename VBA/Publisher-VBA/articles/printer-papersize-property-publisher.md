# <a name="printerpapersize-property-publisher"></a>Свойство Printer.PaperSize (издатель)

Возвращает размер бумаги найдена на вкладке **Параметры публикации и бумаги** в диалоговом окне **Настройка печати** в интерфейсе пользователя Microsoft Publisher (меню " **файл** "). Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **PaperSize**

 переменная _expression_A, представляющий объект **Printer** .


### <a name="return-value"></a>Возвращаемое значение

String


## <a name="remarks"></a>Заметки

Чтобы изменить значение свойства либо **PaperHeight** , либо **PaperWidth** , значение свойства **PaperSize** изменяет текущий.
