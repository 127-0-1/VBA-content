# <a name="printerpaperorientation-property-publisher"></a>Свойство Printer.PaperOrientation (издатель)

Получает или задает принтера, ориентации страницы (книжная или альбомная). Чтение и запись.


## <a name="syntax"></a>Синтаксис

 _выражение_. **PaperOrientation**

 переменная _expression_A, представляющий объект **Printer** .


### <a name="return-value"></a>Возвращаемое значение

PbOrientationType


## <a name="remarks"></a>Заметки

Возможные значения для **PaperOrientation** : **pbOrientationLandscape** (2) и **pbOrientationPortrait** (1).

Настройка свойства **PaperOrientation** соответствует **Ориентация на вкладке **Параметры публикации и бумаги** диалоговое окно **Print Setup** (меню " **файл** "), в пользовательском интерфейсе Microsoft Publisher** .
