# <a name="textrangewordscount-property-publisher"></a>Свойство TextRange.WordsCount (издатель)

Возвращает количество слов в диапазон текста, представленного объектом **TextRange** родительского. Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **WordsCount**

 переменная _expression_A, представляющий объект **TextRange** .


### <a name="return-value"></a>Возвращаемое значение

Длинный
