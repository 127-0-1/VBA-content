# <a name="pblinkedfilestatus-enumeration-publisher"></a>Перечисление PbLinkedFileStatus (издатель)

Указывает состояние файла, связанных с указанным рисунков. 



|**Имя**|**Значение**|**Описание**|
|:-----|:-----|:-----|
| **pbLinkedFileMissing**|2|Больше не удается найти файл по указанному пути.|
| **pbLinkedFileModified**|3|Связанный файл был изменен с момента связь изображения.|
| **pbLinkedFileOK**|1|Файл находится по указанному пути, а не были изменены с момента связь изображения.|
