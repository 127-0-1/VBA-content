# <a name="buildingblockscount-property-publisher"></a>Свойство BuildingBlocks.Count (издатель)

Возвращает значение типа **Long** , представляющее количество элементов в указанном семействе сайтов.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Count**

 переменная _expression_A, представляет собой объект- **BuildingBlocks** .
