# <a name="pagewidth-property-publisher"></a>Свойство Page.Width (издатель)

Возвращает значение типа **Long** , представляющих ширину (в точках) ячейку, диапазон ячеек или страницы. Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Ширина**

 переменная _expression_A, представляющий объект **Page** .
