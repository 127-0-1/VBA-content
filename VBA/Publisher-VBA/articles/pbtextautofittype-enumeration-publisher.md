# <a name="pbtextautofittype-enumeration-publisher"></a>Перечисление PbTextAutoFitType (издатель)

Представляет, как Microsoft Publisher автоматически изменяет размер шрифта текста и размер объектов TextFrame для лучшего просмотра. 



|**Имя**|**Значение**|**Описание**|
|:-----|:-----|:-----|
| **pbTextAutoFitBestFit**|2|Размер кадра текст изменяется, чтобы вместить текст.|
| **pbTextAutoFitNone**|0|Разрешает переполнения рамки.|
| **pbTextAutoFitShrinkOnOverflow**|1|Шрифт уменьшается, текст умещается в рамку.|
