# <a name="pbverticaltextalignmenttype-enumeration-publisher"></a>Перечисление PbVerticalTextAlignmentType (издатель)

Представляет вертикальное выравнивание текста в текстовом поле. 



|**Имя**|**Значение**|**Описание**|
|:-----|:-----|:-----|
| **pbVerticalTextAlignmentBottom**|2|Текст выравнивается по нижней.|
| **pbVerticalTextAlignmentCenter**|1|Текст выравнивается по центру.|
| **pbVerticalTextAlignmentTop**|0|Текст выравнивается по верхней.|
