# <a name="shapeglow-property-publisher"></a>Свойство Shape.Glow (издатель)

Возвращает объект [GlowFormat](glowformat-object-publisher.md) , представляющий свечения форматирования для фигуры. Только для чтения.


## <a name="syntax"></a>Синтаксис

 _выражение_. **Свечения**

 переменная _expression_A, представляющий объект **фигуры** .


## <a name="return-value"></a>Возвращаемое значение

 **GLOWFORMAT**
