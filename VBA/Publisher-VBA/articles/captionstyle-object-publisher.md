# <a name="captionstyle-object-publisher"></a>Объект CaptionStyle (издатель)

Представляет одного названия стилей. Объект **CaptionStyle** является элементом коллекции **[CaptionStyles](captionstyles-object-publisher.md)** . Коллекция **CaptionStyles** представляет все стили 37 заголовок, доступные в Publisher, необязательно в текущий документ.
