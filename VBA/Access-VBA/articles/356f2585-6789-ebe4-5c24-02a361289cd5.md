
# Application.TempVars Property (Access)

Returns the collection of  **[TempVar](4a0429e6-bcfa-7a8b-7030-6e88c2f1a71d.md)** objects. Read-only **[TempVars](aa81b18b-5e9f-ae44-cbcf-55cf6e37b7f6.md)**.


## Syntax

 _expression_. **TempVars**

 _expression_ A variable that represents an **Application** object.
