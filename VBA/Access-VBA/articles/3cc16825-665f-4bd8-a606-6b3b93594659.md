
# ReturnVars Object (Access)

Represents the collection of  **[ReturnVar](8ad5254d-a249-46ba-ac5d-14943179ce05.md)** objects.


## Remove

The  **RetunVars** collection can store up to 255 **ReturnVar** objects.
