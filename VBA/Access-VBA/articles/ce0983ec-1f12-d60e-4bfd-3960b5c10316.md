
# TempVar.Name Property (Access)

Gets the name of the specified  **[TempVar](4a0429e6-bcfa-7a8b-7030-6e88c2f1a71d.md)**. Read-only **String**.


## Syntax

 _expression_. **Name**

 _expression_ A variable that represents a **TempVar** object.
