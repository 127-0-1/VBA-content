
# NavigationControl.SubForm Property (Access)

Gets or sets the name of the  **[SubForm](60f961fa-dcf4-e1d1-8c50-9e88963f9dec.md)** object used to display forms. Read/write **String**.


## Syntax

 _expression_. **SubForm**

 _expression_ A variable that represents a **NavigationControl** object.
