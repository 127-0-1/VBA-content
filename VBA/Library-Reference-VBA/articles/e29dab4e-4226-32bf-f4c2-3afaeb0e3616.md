
# FileDialog.Item Property (Office)

Gets the text associated with an object. Read-only.


## Syntax

 _expression_. **Item**

 _expression_ Required. A variable that represents a **[FileDialog](71a030f2-3b02-21e1-c156-0514ff5eddb7.md)** object.
