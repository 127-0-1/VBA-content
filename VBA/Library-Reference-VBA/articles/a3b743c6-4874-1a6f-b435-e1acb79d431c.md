
# CustomXMLPrefixMapping.Parent Property (Office)

Gets the  **Parent** object of the **CustomXMLPrefixMapping** object. Read-only.


## Syntax

 _expression_. **Parent**

 _expression_ An expression that returns a **CustomXMLPrefixMapping** object.


### Return Value

Object
