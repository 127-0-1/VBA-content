
# DocumentLibraryVersion.Parent Property (Office)

Gets the  **Parent** object for the **DocumentLibraryVersion** object. Read-only.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents a **DocumentLibraryVersion** object.


### Return Value

Object
