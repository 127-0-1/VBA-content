
# ThemeEffectScheme.Parent Property (Office)

Gets the  **Parent** object for the **ThemeEffectScheme** object. Read-only.


## Syntax

 _expression_. **Parent**

 _expression_ An expression that returns a **ThemeEffectScheme** object.


### Return Value

Object
