
# ThemeFonts.Parent Property (Office)

Gets the  **Parent** object for the **ThemeFonts** object. Read-only.


## Syntax

 _expression_. **Parent**

 _expression_ An expression that returns a **ThemeFonts** object.


### Return Value

Object
