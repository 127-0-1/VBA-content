
# BulletFormat2.StartValue Property (Office)

Gets or sets the beginning value of a bulleted list. Read/write.


## Syntax

 _expression_. **StartValue**

 _expression_ An expression that returns a **BulletFormat2** object.


### Return Value

Integer
