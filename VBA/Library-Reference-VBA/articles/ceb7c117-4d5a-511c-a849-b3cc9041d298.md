
# MetaProperties.Count Property (Office)

Gets a  **Long** indicating the number of items in the **MetaProperties** collection. Read-only.


## Syntax

 _expression_. **Count**

 _expression_ An expression that returns a **MetaProperties** object.


### Return Value

Long
