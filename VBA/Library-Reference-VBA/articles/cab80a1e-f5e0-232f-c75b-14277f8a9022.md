
# ServerPolicy.Parent Property (Office)

Gets the  **Parent** object for the **ServerPolicy** object. Read-only.


## Syntax

 _expression_. **Parent**

 _expression_ An expression that returns a **ServerPolicy** object.


### Return Value

Object
