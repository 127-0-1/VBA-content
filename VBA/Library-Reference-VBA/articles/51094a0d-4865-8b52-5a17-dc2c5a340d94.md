
# CustomXMLValidationError.Text Property (Office)

Gets the text in the  **CustomXMLValidationError** object. Read-only.


## Syntax

 _expression_. **Text**

 _expression_ An expression that returns a **CustomXMLValidationError** object.


### Return Value

String
