
# FileDialog.Title Property (Office)

Sets or gets the title of a file dialog box displayed using the  **FileDialog** object. Read/write.


## Syntax

 _expression_. **Title**

 _expression_ Required. A variable that represents a **[FileDialog](71a030f2-3b02-21e1-c156-0514ff5eddb7.md)** object.
