
# ODSOColumn.Index Property (Office)

Gets a  **Long** representing the index number for an **ODSOColumn** object in the collection. Read-only.


## Syntax

 _expression_. **Index**

 _expression_ A variable that represents an **ODSOColumn** object.


### Return Value

Long
