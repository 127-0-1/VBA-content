
# PolicyItem.Id Property (Office)

Gets the ID of a policy item.  **PolicyItem** objects are contained in **ServerPolicy** objects. Read-only.


## Syntax

 _expression_. **Id**

 _expression_ An expression that returns a **PolicyItem** object.


### Return Value

String
