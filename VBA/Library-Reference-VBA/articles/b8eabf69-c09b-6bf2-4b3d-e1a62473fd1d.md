
# CustomXMLSchema.Parent Property (Office)

Gets the  **Parent** object for the **CustomXMLSchema** object. Read-only.


## Syntax

 _expression_. **Parent**

 _expression_ An expression that returns a **CustomXMLSchema** object.


### Return Value

Object
