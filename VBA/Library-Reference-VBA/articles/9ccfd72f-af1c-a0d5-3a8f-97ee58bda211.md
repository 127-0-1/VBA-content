
# Signature.Setup Property (Office)

Gets a  **SignatureSetup** object that provides access to various properties of a signature packet. Read-only.


## Syntax

 _expression_. **Setup**

 _expression_ An expression that returns a **Signature** object.


### Return Value

SignatureSetup
