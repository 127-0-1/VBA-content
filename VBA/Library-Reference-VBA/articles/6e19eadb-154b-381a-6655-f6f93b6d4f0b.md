
# CustomXMLPart.Id Property (Office)

Gets a  **String** containing the GUID assigned to the current **CustomXMLPart** object. Read-only.


## Syntax

 _expression_. **Id**

 _expression_ An expression that returns a **CustomXMLPart** object.


### Return Value

String
