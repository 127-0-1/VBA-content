
# Ruler2.TabStops Property (Office)

Gets a  **TabStops2** collection that represents the tab stops for the specified text. Read-only.


## Syntax

 _expression_. **TabStops**

 _expression_ An expression that returns a **Ruler2** object.


### Return Value

TabStops2
