
# DocumentLibraryVersion.Delete Method (Office)

Removes a document library version from the  **DocumentLibraryVersions** collection.


## Syntax

 _expression_. **Delete**

 _expression_ Required. A variable that represents a **[DocumentLibraryVersion](ac13975d-4f91-1fc5-5b0a-94b21309ffb7.md)** object.
