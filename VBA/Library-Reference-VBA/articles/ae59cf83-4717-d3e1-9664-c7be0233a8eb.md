
# Font2.Reflection Property (Office)

Gets a value specifying the type of reflection format for the selection of text. Read-only.


## Syntax

 _expression_. **Reflection**

 _expression_ An expression that returns a **Font2** object.


### Return Value

ReflectionFormat
