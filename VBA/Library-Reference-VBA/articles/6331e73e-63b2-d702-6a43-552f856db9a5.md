
# RulerLevel2.Parent Property (Office)

Gets the  **Parent** object for the **RulerLevel2** object. Read-only.


## Syntax

 _expression_. **Parent**

 _expression_ An expression that returns a **RulerLevel2** object.


### Return Value

Object
