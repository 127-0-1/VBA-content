
# WorkflowTask.AssignedTo Property (Office)

Gets the name of the person that the workflow task is assigned to. Read-only.


## Syntax

 _expression_. **AssignedTo**

 _expression_ An expression that returns a **WorkflowTask** object.


### Return Value

String
