
# DataTable.Delete Method (PowerPoint)

Deletes the object.


## Syntax

 _expression_. **Delete**

 _expression_ A variable that represents a **[DataTable](eaa7cdda-e374-7d19-47a6-87e4458fc244.md)** object.
