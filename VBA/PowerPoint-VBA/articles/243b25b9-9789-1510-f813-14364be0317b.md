
# Row.Delete Method (PowerPoint)

Deletes the specified  **Row** object.


## Syntax

 _expression_. **Delete**

 _expression_ A variable that represents a **Row** object.


## Remarks

Attempting to delete the only existing row in a table will result in a run-time error.
