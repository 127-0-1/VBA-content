
# DropLines.Name Property (PowerPoint)

Returns the name of the object. Read-only  **String**.


## Syntax

 _expression_. **Name**

 _expression_ A variable that represents a **[DropLines](b13b58c3-d00d-16d2-16ef-bcd3cae347c5.md)** object.
