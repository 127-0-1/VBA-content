
# AxisTitle.Parent Property (PowerPoint)

Returns the parent for the specified object. Read-only  **Object**.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents an **[AxisTitle](8eddc95c-2353-43fa-c055-ee76de28009d.md)** object.
