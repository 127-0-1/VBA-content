
# MediaFormat.IsEmbedded Property (PowerPoint)

Returns whether the media file is embedded. Read-only.


## Syntax

 _expression_. **IsEmbedded**

 _expression_ An expression that returns a **MediaFormat** object.


### Return Value

Boolean
