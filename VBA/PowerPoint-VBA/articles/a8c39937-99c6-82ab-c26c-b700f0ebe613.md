
# View.MediaControlsWidth Property (PowerPoint)

Returns the width, in points, of the media control bounding box. Read-only.


## Syntax

 _expression_. **MediaControlsWidth**

 _expression_ A variable that represents a **View** object.


### Return Value

Single
