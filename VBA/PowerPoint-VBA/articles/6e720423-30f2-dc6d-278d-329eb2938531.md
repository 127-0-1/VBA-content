
# ChartArea.Parent Property (PowerPoint)

Returns the parent for the specified object. Read-only  **Object**.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents a **[ChartArea](2c8bd84e-18e7-6417-de4d-d643064e20f5.md)** object.
