
# UpBars.Parent Property (PowerPoint)

Returns the parent for the specified object. Read-only  **Object**.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents an **[UpBars](8a176f01-01a6-86bc-a69b-29763ebb1481.md)** object.
