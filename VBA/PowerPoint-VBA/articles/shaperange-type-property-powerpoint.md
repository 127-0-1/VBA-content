# ShapeRange.Type Property (PowerPoint)

Represents the type of shape or shapes in a range of shapes. Read-only.


## Syntax

 _expression_. **Type**

 _expression_ A variable that represents a **ShapeRange** object.


### Return Value

MsoShapeType
