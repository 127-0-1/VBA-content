
# TextRange2.Start Property (PowerPoint)

Gets a  **Long** value indicating the starting point of the specified text range. Read-only.


## Syntax

 _expression_. **Start**

 _expression_ An expression that returns a **TextRange2** object.


### Return Value

Long
