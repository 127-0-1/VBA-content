
# SeriesLines.Name Property (PowerPoint)

Returns the name of the object. Read-only  **String**.


## Syntax

 _expression_. **Name**

 _expression_ A variable that represents a **[SeriesLines](5d953ed4-ca16-3cb3-ba8f-1742e4a56cb6.md)** object.
