
# Trendline.ClearFormats Method (PowerPoint)

Clears the formatting of the object.


## Syntax

 _expression_. **ClearFormats**

 _expression_ A variable that represents a **[Trendline](74755c19-0a7d-cbbf-857e-78740adf6aa4.md)** object.
