
# TextFrame2.ThreeD Property (PowerPoint)

Returns a  **[ThreeDFormat](d6eb7b36-57df-727e-fc5b-50b8c4790c1c.md)** object that represents the three-dimensional formatting of the parent shape. Read-only.


## Syntax

 _expression_. **ThreeD**

 _expression_ An expression that returns a **TextFrame2** object.


### Return Value

ThreeDFormat
