
# UpBars.Name Property (PowerPoint)

Returns the name of the object. Read-only  **String**.


## Syntax

 _expression_. **Name**

 _expression_ A variable that represents an **[UpBars](8a176f01-01a6-86bc-a69b-29763ebb1481.md)** object.
