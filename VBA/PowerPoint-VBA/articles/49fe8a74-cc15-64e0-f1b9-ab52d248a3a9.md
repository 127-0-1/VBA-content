
# SeriesLines.Delete Method (PowerPoint)

Deletes the object.


## Syntax

 _expression_. **Delete**

 _expression_ A variable that represents a **[SeriesLines](5d953ed4-ca16-3cb3-ba8f-1742e4a56cb6.md)** object.
