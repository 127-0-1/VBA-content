
# Chart.SideWall Property (PowerPoint)

Returns a  **[Walls](b2288a5f-efec-84b4-9a40-d62d61196ac8.md)** object that allows the user to individually format the side wall of a 3-D chart. Read-only.


## Syntax

 _expression_. **SideWall**

 _expression_ A variable that represents a **[Chart](3fcf082f-9f58-f67d-1061-e7f37e30fbcd.md)** object.
