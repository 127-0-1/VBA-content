
# DataLabels.Count Property (PowerPoint)

Returns the number of objects in the collection. Read-only  **Long**.


## Syntax

 _expression_. **Count**

 _expression_ A variable that represents a **[DataLabels](a0d0b0ec-6a12-9a5c-1026-1e1d85e488fa.md)** object.
