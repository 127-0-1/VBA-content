
# Axes.Count Property (PowerPoint)

Returns the number of objects in the collection. Read-only  **Long**.


## Syntax

 _expression_. **Count**

 _expression_ A variable that represents an **[Axes](71f1e1fc-7086-a84e-1e05-6fa50597b49b.md)** object.
