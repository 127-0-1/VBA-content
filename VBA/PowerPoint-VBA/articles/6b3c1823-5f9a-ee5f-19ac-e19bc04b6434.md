
# Trendline.Name Property (PowerPoint)

Returns or sets name of the object. Read/write  **String**.


## Syntax

 _expression_. **Name**

 _expression_ A variable that represents a **[Trendline](74755c19-0a7d-cbbf-857e-78740adf6aa4.md)** object.
