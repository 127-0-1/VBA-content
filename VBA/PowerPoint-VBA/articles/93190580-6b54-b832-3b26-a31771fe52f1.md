
# Walls.Parent Property (PowerPoint)

Returns the parent for the specified object. Read-only  **Object**.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents a **[Walls](b2288a5f-efec-84b4-9a40-d62d61196ac8.md)** object.
