
# Trendlines.Parent Property (PowerPoint)

Returns the parent for the specified object. Read-only  **Object**.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents a **[Trendlines](8ac46695-aae0-3611-ebf7-c7339ea733ab.md)** object.
