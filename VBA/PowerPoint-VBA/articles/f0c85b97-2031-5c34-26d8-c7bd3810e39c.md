
# AxisTitle.Name Property (PowerPoint)

Returns a  **String** value that represents the name of the object.


## Syntax

 _expression_. **Name**

 _expression_ A variable that represents an **[AxisTitle](8eddc95c-2353-43fa-c055-ee76de28009d.md)** object.
