
# ChartGroups.Count Property (PowerPoint)

Returns the number of objects in the collection. Read-only  **Long**.


## Syntax

 _expression_. **Count**

 _expression_ A variable that represents a **[ChartGroups](2db874db-91af-0b1e-7496-92a8443caade.md)** object.
