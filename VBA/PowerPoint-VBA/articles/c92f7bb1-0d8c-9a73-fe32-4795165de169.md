
# LegendEntry.Format Property (PowerPoint)

Returns the line, fill, and effect formatting for the object. Read-only  **[ChartFormat](bba095c6-2abf-eb14-10d4-35686c06941c.md)**.


## Syntax

 _expression_. **Format**

 _expression_ A variable that represents a **[LegendEntry](c92ddccd-92a3-bec9-cdcd-efd82c77706b.md)** object.
