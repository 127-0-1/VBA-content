
# LegendKey.ClearFormats Method (PowerPoint)

Clears the formatting of the object.


## Syntax

 _expression_. **ClearFormats**

 _expression_ A variable that represents a **[LegendKey](98e8b9c3-b53e-9595-9389-6f92a6d730f4.md)** object.
