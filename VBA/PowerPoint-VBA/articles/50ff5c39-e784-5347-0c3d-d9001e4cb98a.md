
# Walls.Name Property (PowerPoint)

Returns the name of the object. Read-only  **String**.


## Syntax

 _expression_. **Name**

 _expression_ A variable that represents a **[Walls](b2288a5f-efec-84b4-9a40-d62d61196ac8.md)** object.
