
# CustomLayout.Application Property (PowerPoint)

Returns an  **[Application](978c2b99-4271-b953-4283-73b5f3d96f41.md)** object that represents the creator of the specified object. Read-only.


## Syntax

 _expression_. **Application**

 _expression_ A variable that represents a **CustomLayout** object.


### Return Value

Application
