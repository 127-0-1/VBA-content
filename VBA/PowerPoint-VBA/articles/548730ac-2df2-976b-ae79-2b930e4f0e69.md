
# MediaFormat.SampleHeight Property (PowerPoint)

Returns the resolution height of the sample media. Read-only.


## Syntax

 _expression_. **SampleHeight**

 _expression_ An expression that returns a **MediaFormat** object.


### Return Value

Long
