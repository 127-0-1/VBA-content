
# Floor.Name Property (PowerPoint)

Returns the name of the object. Read-only  **String**.


## Syntax

 _expression_. **Name**

 _expression_ A variable that represents a **[Floor](ed9ff3d1-8001-840c-d26e-7513ebe73ae9.md)** object.
