
# Chart.SetBackgroundPicture Method (PowerPoint)

Sets the background graphic for a chart.


## Syntax

 _expression_. **SetBackgroundPicture**( **_FileName_** )

 _expression_ A variable that represents a **[Chart](3fcf082f-9f58-f67d-1061-e7f37e30fbcd.md)** object.


### Parameters



|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _FileName_|Required|**String**|The name of the file for the graphic.|
