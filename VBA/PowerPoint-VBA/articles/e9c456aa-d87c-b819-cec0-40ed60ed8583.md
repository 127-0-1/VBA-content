
# DropLines.Parent Property (PowerPoint)

Returns the parent for the specified object. Read-only  **Object**.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents a **[DropLines](b13b58c3-d00d-16d2-16ef-bcd3cae347c5.md)** object.
