
# Chart.Refresh Method (PowerPoint)

Causes the specified chart to be redrawn immediately.


## Syntax

 _expression_. **Refresh**

 _expression_ A variable that represents a **[Chart](3fcf082f-9f58-f67d-1061-e7f37e30fbcd.md)** object.
