
# CustomLayout.HeadersFooters Property (PowerPoint)

Returns a  **[HeadersFooters](5fb10c90-0611-e797-836b-3f18b273af04.md)** collection that represents the header, footer, date and time, and slide number associated with the specified custom layout.


## Syntax

 _expression_. **HeadersFooters**

 _expression_ A variable that represents a **CustomLayout** object.


### Return Value

HeadersFooters
