
# AxisTitle.Width Property (PowerPoint)

Returns a  **Double** value that represents the width, in points, of the object. Read-only.


## Syntax

 _expression_. **Width**

 _expression_ A variable that represents an **AxisTitle** object.


### Return Value

Double
