
# Interior.Parent Property (PowerPoint)

Returns the parent for the specified object. Read-only  **Object**.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents an **[Interior](8b971d83-253e-50c9-5838-9f07f0a34cbd.md)** object.
