
# LegendKey.Height Property (PowerPoint)

Returns the height, in points, of the object. Read-only  **Double**.


## Syntax

 _expression_. **Height**

 _expression_ A variable that represents a **[LegendKey](98e8b9c3-b53e-9595-9389-6f92a6d730f4.md)** object.
