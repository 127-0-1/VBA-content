
# Floor.Parent Property (PowerPoint)

Returns the parent for the specified object. Read-only  **Object**.


## Syntax

 _expression_. **Parent**

 _expression_ A variable that represents a **[Floor](ed9ff3d1-8001-840c-d26e-7513ebe73ae9.md)** object.
