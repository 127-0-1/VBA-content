
# Model.ModelRelationships Property (Excel)

Returns a collection of relationships between Data Model tables. Read-only


## Syntax

 _expression_ . **ModelRelationships**

 _expression_ A variable that represents a object.


## Property value

 **MODELRELATIONSHIPS**
