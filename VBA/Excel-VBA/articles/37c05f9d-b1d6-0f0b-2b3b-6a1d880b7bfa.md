
# SlicerPivotTables.Count Property (Excel)

Returns the number of objects in the collection.


## Syntax

 _expression_ . **Count**

 _expression_ A variable that returns a **[SlicerPivotTables](8302dc8a-3845-12b0-f88e-761f104f1dcc.md)** object.


### Return Value

 **Long**
