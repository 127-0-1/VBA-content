
# SlicerItem.Value Property (Excel)

Returns the value of the slicer item. Read-only.


## Syntax

 _expression_ . **Value**

 _expression_ A variable that represents a **[SlicerItem](cb93cd82-fc3a-f6b7-ae64-db6312db649d.md)** object.


### Return Value

String
