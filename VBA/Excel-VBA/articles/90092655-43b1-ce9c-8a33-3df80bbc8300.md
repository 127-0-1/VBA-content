
# DownBars.Format Property (Excel)

Returns the  **[ChartFormat](edac71b7-ed38-6658-2cbf-6493dc1ad3ed.md)** object. Read-only.


## Syntax

 _expression_ . **Format**

 _expression_ A variable that represents a **DownBars** object.
