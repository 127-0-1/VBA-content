
# Characters.Font Property (Excel)

Returns a  **[Font](f4788ba4-1c4c-2f03-4d73-194bc9316825.md)** object that represents the font of the specified object.


## Syntax

 _expression_ . **Font**

 _expression_ A variable that represents a **Characters** object.
