
# ThreeDFormat.ContourColor Property (Excel)

Returns the contour color for a  **ThreeDFormat** object. Read-only **ColorFormat** .


## Syntax

 _expression_ . **ContourColor**

 _expression_ A variable that represents a **ThreeDFormat** object.
