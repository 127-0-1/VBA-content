
# Chart.ChartColor Property (Excel)

Returns or sets an  **Integer** that represents the color scheme for the chart. Read-write.


## Syntax

 _expression_ . **ChartColor**

 _expression_ A variable that represents a **Chart** object.


## Property value

 **VARIANT**
