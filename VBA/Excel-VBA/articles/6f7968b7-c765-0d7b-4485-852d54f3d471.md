
# Model.ModelFormatScientificNumber Property (Excel)

Returns a [ModelFormatScientificNumber](0099a473-0848-05ad-abe5-b36b70d4a2da.md) object that represents formatting of type scientific number in the data model. Read-only.


## Syntax

 _expression_ . **ModelFormatScientificNumber**

 _expression_ A variable that represents a **Model** object.
