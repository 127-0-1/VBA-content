
# SlicerCache.ClearDateFilter Method (Excel)

Clears the filter for a timeline (date filter).


## Syntax

 _expression_ . **ClearDateFilter**

 _expression_ A variable that represents a[SlicerCache Object (Excel)](6e6533e3-0503-a1d3-9ecd-f7997233565f.md) object.


### Return value

 **VOID**
