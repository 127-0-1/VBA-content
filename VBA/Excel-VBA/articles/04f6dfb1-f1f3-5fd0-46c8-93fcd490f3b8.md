
# DataFeedConnection.CancelRefresh Method (Excel)

Cancels a refresh operation on a data feed connection.


## Syntax

 _expression_ . **CancelRefresh**

 _expression_ A variable that represents a[DataFeedConnection Object (Excel)](2ccb242b-28d5-3baf-78be-aa8f7478f4b6.md) object.


### Return value

 **VOID**
