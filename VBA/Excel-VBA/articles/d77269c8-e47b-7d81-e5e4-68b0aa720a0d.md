
# Application.FlashFillMode Property (Excel)

 **True** if the Flash Fill feature is enabled. **Boolean** Read/Write


## Syntax

 _expression_ . **FlashFillMode**

 _expression_ A variable that represents a **Application** object.


## Property value

 **BOOL**
