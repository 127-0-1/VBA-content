
# DataFeedConnection.CommandType Property (Excel)

Returns or sets the command string for the specified data source.  **Variant** Read/Write


## Syntax

 _expression_ . **CommandType**

 _expression_ A variable that represents a[DataFeedConnection Object (Excel)](2ccb242b-28d5-3baf-78be-aa8f7478f4b6.md) object.


## Property value

 **XLCMDTYPE**
