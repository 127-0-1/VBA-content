
# Model.ModelFormatCurrency Property (Excel)

Returns a [ModelFormatCurrency](acb863b6-c188-5ed3-afe4-5e1ab6bb20bf.md) object that represents formatting of type currency in the data model. Read-only .


## Syntax

 _expression_ . **ModelFormatCurrency**

 _expression_ A variable that represents a **Model** object.
