
# Slicer.Width Property (Excel)

Returns or sets the width of the specified slicer, in points. Read/write.


## Syntax

 _expression_ . **Width**

 _expression_ A variable that represents a **[Slicer](577be0f6-4eda-0093-8899-097f3c900383.md)** object.


### Return Value

Double
