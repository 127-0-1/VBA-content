
# TableObject.Delete Method (Excel)

Deletes the  **TableObject** object.


## Syntax

 _expression_ . **Delete**

 _expression_ A variable that represents a[ModelTable Object (Excel)](c853beb6-f2e7-dda0-b33a-8110a6c23de8.md) object.


### Return value

 **VOID**
