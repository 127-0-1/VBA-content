
# OlkCommandButton.PictureAlignment Property (Outlook)

Returns or sets an  **[OlPictureAlignment](69b9407f-a86a-393d-11ac-5272be88de6d.md)** constant that specifies the alignment of the image and the text on the button. Read/write.


## Syntax

 _expression_ . **PictureAlignment**

 _expression_ A variable that represents an **OlkCommandButton** object.
