
# RemoteItem.UnRead Property (Outlook)

Returns or sets a  **Boolean** value that is **True** if the Outlook item has not been opened (read). Read/write.


## Syntax

 _expression_ . **UnRead**

 _expression_ A variable that represents a **RemoteItem** object.
