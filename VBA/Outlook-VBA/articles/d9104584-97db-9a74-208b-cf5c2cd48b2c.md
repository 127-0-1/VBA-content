
# StorageItem.CreationTime Property (Outlook)

Returns a  **DateTime** value that indicates the creation time for the **[StorageItem](41776bc3-b838-2755-fd6b-3b5012fb9ae5.md)** . Read-only.


## Syntax

 _expression_ . **CreationTime**

 _expression_ A variable that represents a **StorageItem** object.
