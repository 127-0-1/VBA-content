
# ContactItem.SelectedMailingAddress Property (Outlook)

Returns or sets an  **[OlMailingAddress](1a0822be-3c4c-f162-8ab7-12edb20a0949.md)** constant indicating the type of the mailing address for the contact. Read/write.


## Syntax

 _expression_ . **SelectedMailingAddress**

 _expression_ A variable that represents a **ContactItem** object.
