
# Row.Parent Property (Outlook)

Returns the parent object of the  **[Row](06db3fa4-1649-48bf-3b86-ffdf99a47305.md)** object. Read-only.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents a **Row** object.
