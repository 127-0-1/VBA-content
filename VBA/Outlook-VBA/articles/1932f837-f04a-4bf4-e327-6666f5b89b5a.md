
# TableView.HideReadingPaneHeaderInfo Property (Outlook)

Returns or sets a  **Boolean** value that determines whether the header for an Outlook item is displayed in the Reading Pane for the **[TableView](026e27f8-1655-060d-e8cc-87eaaf4f1510.md)** object. Read/write.


## Syntax

 _expression_ . **HideReadingPaneHeaderInfo**

 _expression_ A variable that represents a **TableView** object.
