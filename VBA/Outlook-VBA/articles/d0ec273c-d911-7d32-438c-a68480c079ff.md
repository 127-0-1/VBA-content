
# SharingItem.Size Property (Outlook)

Returns a  **Long** indicating the size (in bytes) of the **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** . Read-only.


## Syntax

 _expression_ . **Size**

 _expression_ A variable that represents a **SharingItem** object.
