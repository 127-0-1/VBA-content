
# OlkTimeControl.Value Property (Outlook)

Returns or sets a  **Variant** that represents the content of the control. Read/write.


## Syntax

 _expression_ . **Value**

 _expression_ A variable that represents an **OlkTimeControl** object.


## Remarks

The value is the same as the value of  **[Text](bf9c97af-65af-655a-642f-17af6513c670.md)** , except that this property is **Variant** and **Text** is strongly typed as **String** .
