
# SharingItem.Saved Property (Outlook)

Returns a  **Boolean** value that is **true** if the **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** has not been modified since the last save. Read-only.


## Syntax

 _expression_ . **Saved**

 _expression_ A variable that represents a **SharingItem** object.
