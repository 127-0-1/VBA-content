
# SharingItem.IsMarkedAsTask Property (Outlook)

Returns a  **Boolean** value that indicates whether the **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** is marked as a task. Read-only.


## Syntax

 _expression_ . **IsMarkedAsTask**

 _expression_ An expression that returns a **SharingItem** object.
