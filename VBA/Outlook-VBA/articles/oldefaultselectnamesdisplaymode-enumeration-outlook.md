# OlDefaultSelectNamesDisplayMode Enumeration (Outlook)

Specifies the default caption, the number of buttons, the button labels, and the address lists to display in the  **Select Names** dialog box without requiring a resource file for any localized caption and labels.



| <strong>Name</strong>                    | <strong>Value</strong> | <strong>Description</strong>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
|:-----------------------------------------|:-----------------------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| <strong>olDefaultDelegates</strong>      | 6                      | Displays one edit box for To recipients, uses localized string representing "Add" for the To button, and localized string representing "Add Users" for the caption.  <strong>CcLabel</strong> and <strong>BccLabel</strong> are set to an empty string. Sets <strong><a href="selectnamesdialog-allowmultipleselection-property-outlook.md" data-raw-source="[SelectNamesDialog.AllowMultipleSelection](selectnamesdialog-allowmultipleselection-property-outlook.md)">SelectNamesDialog.AllowMultipleSelection</a></strong> to <strong>True</strong> and <strong><a href="selectnamesdialog-numberofrecipientselectors-property-outlook.md" data-raw-source="[SelectNamesDialog.NumberOfRecipientSelectors](selectnamesdialog-numberofrecipientselectors-property-outlook.md)">SelectNamesDialog.NumberOfRecipientSelectors</a></strong> to <strong>olTo</strong>. |
| <strong>olDefaultMail</strong>           | 1                      | Displays three edit boxes for To, Cc, and Bcc recipients, uses localized strings representing "To", "Cc", and "Bcc" for To, Cc, and Bcc buttons, and localized string representing "Select Names" for the caption. Sets  <strong>AllowMultipleSelection</strong> to <strong>True</strong> and <strong>NumberOfRecipientSelectors</strong> to <strong>olToCcBcc</strong>.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| <strong>olDefaultMeeting</strong>        | 2                      | Displays three edit boxes for Required, Optional, and Resource recipients, uses localized strings representing "Required", "Optional", and "Resources" for the To, Cc, and Bcc buttons, and localized string representing "Select Attendees and Resources" for the caption. Sets  <strong>AllowMultipleSelection</strong> to <strong>True</strong> and <strong>NumberOfRecipientSelectors</strong> to <strong>olToCcBcc</strong>.                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| <strong>olDefaultMembers</strong>        | 5                      | Displays one edit box for To recipients, uses localized string representing "To" for the To button, and localized string representing "Select Members" for caption.  <strong>CcLabel</strong> and <strong>BccLabel</strong> are set to an empty string. Sets <strong>AllowMultipleSelection</strong> to <strong>True</strong> and <strong>NumberOfRecipientSelectors</strong> to <strong>olTo</strong>.                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| <strong>olDefaultPickRooms</strong>      | 8                      | Displays one edit box for Resource recipients, uses localized string representing "Rooms" for To button, and localized string representing "Select Rooms" for caption.  <strong>CcLabel</strong> and <strong>BccLabel</strong> are set to an empty string. Sets <strong>AllowMultipleSelection</strong> to <strong>True</strong> and <strong>NumberOfRecipientSelectors</strong> to *<strong><em>olShowTo.  *</em>InitialDisplayList</strong> is set to the Global Address List.                                                                                                                                                                                                                                                                                                                                                                                    |
| <strong>olDefaultSharingRequest</strong> | 4                      | Displays one edit box for To recipients, uses localized string representing "To" for To button, and localized string representing "Select Names" for caption.  <strong>CcLabel</strong> and <strong>BccLabel</strong> are set to an empty string. Sets <strong>AllowMultipleSelection</strong> to <strong>True</strong> and <strong>NumberOfRecipientSelectors</strong> to <strong>olTo</strong>.                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| <strong>olDefaultSingleName</strong>     | 7                      | Displays no edit boxes for recipients, uses localized string representing "Select Name" for caption.  <strong>ToLabel</strong>,  <strong>CcLabel</strong>, and  <strong>Bcclabel</strong> are set to an empty string. Sets <strong>AllowMultipleSelection</strong> to <strong>False</strong> and <strong>NumberOfRecipientSelectors</strong> to <strong>olNone</strong>.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| <strong>olDefaultTask</strong>           | 3                      | Displays one edit box for To recipients, uses localized string representing "To" for To button, and localized string representing "Select Task Recipient" for caption.  <strong>CcLabel</strong> and <strong>BccLabel</strong> are set to an empty string. Sets <strong>AllowMultipleSelection</strong> to <strong>True</strong> and <strong>NumberOfRecipientSelectors</strong> to <strong>olTo</strong>.                                                                                                                                                                                                                                                                                                                                                                                                                                                          |
