
# OlkDateControl.Change Event (Outlook)

Occurs when the  **[Value](df2c96d4-42d4-fd33-a55b-2162f65069b7.md)** property changes.


## Syntax

 _expression_ . **Change**

 _expression_ A variable that represents an **OlkDateControl** object.
