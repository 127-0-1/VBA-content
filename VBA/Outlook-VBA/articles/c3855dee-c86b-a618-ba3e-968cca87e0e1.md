
# TableView.Apply Method (Outlook)

Applies the  **[TableView](026e27f8-1655-060d-e8cc-87eaaf4f1510.md)** object to the current view.


## Syntax

 _expression_ . **Apply**

 _expression_ A variable that represents a **TableView** object.
