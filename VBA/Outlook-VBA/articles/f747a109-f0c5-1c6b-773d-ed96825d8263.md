
# SharingItem.ExpiryTime Property (Outlook)

Returns or sets a  **Date** indicating the date and time at which the **[SharingItem](63dd3451-44f3-7cc4-c6e2-7dad5835a7d2.md)** becomes invalid and can be deleted. Read/write.


## Syntax

 _expression_ . **ExpiryTime**

 _expression_ A variable that represents a **SharingItem** object.
