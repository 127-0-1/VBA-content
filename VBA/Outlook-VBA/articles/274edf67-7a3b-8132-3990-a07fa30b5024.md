
# CalendarView.Apply Method (Outlook)

Applies the  **[CalendarView](37e078b9-9fc6-5894-b043-06d7257666a8.md)** object to the current view.


## Syntax

 _expression_ . **Apply**

 _expression_ A variable that represents a **CalendarView** object.
