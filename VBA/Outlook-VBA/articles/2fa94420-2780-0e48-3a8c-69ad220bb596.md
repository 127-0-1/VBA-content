
# CardView.Apply Method (Outlook)

Applies the  **[CardView](cdac229b-f2b6-9ecb-e1a7-b53509426570.md)** object to the current view.


## Syntax

 _expression_ . **Apply**

 _expression_ A variable that represents a **CardView** object.
