
# AssignToCategoryRuleAction.Parent Property (Outlook)

Returns the parent  **Object** of the specified object. Read-only.


## Syntax

 _expression_ . **Parent**

 _expression_ A variable that represents an **AssignToCategoryRuleAction** object.


## Remarks

The parent of the  **[AssignToCategoryRuleAction](402f4742-72ba-2559-4e4c-e2b8248cd7f6.md)** object is the **[RuleAction](6451788f-e5ed-239c-a34d-b564b52d8955.md)** object.
