
# CalendarView.BoldDatesWithItems Property (Outlook)

Returns or sets a  **Boolean** value that indicates whether days that contain Outlook items should be displayed in bold within the Date Navigator for the **[CalendarView](37e078b9-9fc6-5894-b043-06d7257666a8.md)** object. Read/write.


## Syntax

 _expression_ . **BoldDatesWithItems**

 _expression_ A variable that represents a **CalendarView** object.
