
# OlkTimeControl.Style Property (Outlook)

Returns or sets an  **[OlTimeStyle](82c4d063-29f2-d7c8-44ff-8b4aca912855.md)** constant that specifies the style of the time control. Read/write.


## Syntax

 _expression_ . **Style**

 _expression_ A variable that represents an **OlkTimeControl** object.


## Remarks

The default value is  **olTimeStyleTimeOnly** .
