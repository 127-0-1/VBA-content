
# CardView.ShowEmptyFields Property (Outlook)

Returns or sets a  **Boolean** value that determines if empty fields are displayed for cards in the **[CardView](cdac229b-f2b6-9ecb-e1a7-b53509426570.md)** object. Read/write.


## Syntax

 _expression_ . **ShowEmptyFields**

 _expression_ A variable that represents a **CardView** object.
