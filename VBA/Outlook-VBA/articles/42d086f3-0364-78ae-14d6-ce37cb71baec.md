
# RemoteItem.Copy Method (Outlook)

Creates another instance of an object.


## Syntax

 _expression_ . **Copy**

 _expression_ A variable that represents a **[RemoteItem](6302aaff-cdcf-4d86-60f1-4bed15540d9f.md)** object.
