
# PostItem.IsMarkedAsTask Property (Outlook)

Returns a  **Boolean** value that indicates whether the **[PostItem](de44065d-4e93-315a-279f-7b92f09c0465.md)** is marked as a task. Read-only.


## Syntax

 _expression_ . **IsMarkedAsTask**

 _expression_ An expression that returns a **PostItem** object.
